# 修四边形 (rect*repair) · 组织深度介绍

> 基于 EP19 播客及逐字稿的深度分析。
> 节目时长约 90 分钟，由重轻主持，Joanna 和 Tianqi 为嘉宾。

---

## 一、组织速写

**修四边形 (rect\*repair)** 是一个驻扎在上海的艺术游戏 collective，由 Joanna（阿久）、Tianqi（天琦）、Viola 和岳敏四人组成。他们专注于 **pervasive game（渗透现实游戏）**——在真实世界的街道、俱乐部、美术馆、办公室和高铁上设计游戏规则，让陌生人以意想不到的方式连接在一起。

**一句话定义**：他们不"造"四边形，而是"修"——不声称能重建现实，而是在现实的裂缝中撬开一点，创造一个短暂的魔法圈，让人踏入后处于真实与虚拟的模糊中间地带。

---

## 二、命名哲学："修"而非"造"

> "四边形是现实生活中比较常见的一个形状。楼房、窗子、门、屏幕也都会优先用四边形来做。修四边形就是想从一个现实体的形态撬开一点，让你进到一个不那么真实的……用游戏去创造一个小的魔法圈，让人踏入这个魔法圈之后，你就处在一个现实跟虚拟的模糊中间地带上。" —— Tianqi

> "造不了四边形，没有这个能力重建一个现实世界。" —— Tianqi

> "修这个字比较有美感吧。" —— Joanna

**三个关键词**：

| 关键词 | 含义 |
|--------|------|
| **修** (repair) | 不是从零创造，而是在现存现实中修补、撬开、改造 |
| **四边形** (rectangle) | 现实世界的基本形态——建筑、屏幕、窗户、门，一切规训我们的框架 |
| **魔法圈** (magic circle) | 游戏设计核心概念：规则划定的临时空间，在其中现实暂时悬置 |

**深层含义**：修四边形的实践本质上是一种**修补术（bricolage）**——不是带着蓝图去建造一个新世界，而是用手头已有的材料（上海的街道、商场、俱乐部、办公室、微信群）去撬动和变形。这种姿态同时是谦逊的（"造不了"）和激进的（"撬开"）。

---

## 三、成立与演化

### 起源

- 最初 3 个人，没有长期规划——"一开始我们也没有说要把它做成一个长期的项目"
- 自我定位为 **collective（艺术小组）**，类比乐队而非公司
- Joanna 是发起者：从互娱大厂离职 → 做媒体艺术 → 参加雅典 **Trust in Play camp** → 转折

### Joanna 的个人转折

> "当时不是刚从大厂离职，觉得电子游戏已经没有什么特别，就觉得特别绝望，又加上这个电子阳痿，又加上被大厂折磨，觉得电子游戏没啥意思。"

> "去了这个 camp……发现其实游戏在很多其他领域，它其实可以再跟很多东西进行交互的。然后我可能发现，诶，玩还是特别好玩，就 play 还是一个特别好的东西。"

> "因为它（电子游戏）太赚钱了，被太多东西绑架了。"

**转折路径**：

```
互娱大厂（被异化）
    ↓
媒体艺术（探索出口）
    ↓
雅典 Trust in Play camp ★ 关键转折
    ↓
Warp and Weft（第一个 pervasive game）
    ↓
修四边形 collective 形成
```

### 组织参考：MSCHF

Joanna 明确提到修四边形的参考对象是 **MSCHF**——一个纽约的创意集体，做互联网艺术项目、卖产品、做线下活动、不被 gallery 也不被商业单一逻辑绑架。

> "他们又做了一些你感觉像是有 message 和有调性在那的艺术项目，同时它还是 toC 的，它不需要被当代艺术的这个东西绑架……同时也卖包……同时也去做他们想要的这些互联网艺术。"

这揭示了修四边形的自我定位：**介于艺术、商业、社群、互联网之间的杂交体**。

## 四、核心实践：Pervasive Game（渗透现实游戏）

### 理论根源

Joanna 引用了学术著作 **《Pervasive Games》**，将修四边形的实践放在一个更广阔的谱系中：

| 子类型 | 说明 | 修四边形案例 |
|--------|------|------------|
| **LARP** (实况角色扮演) | 参与者在现实空间中扮演角色 | Club with no entry、Dancefloor Assassination |
| **ARG** (另类现实游戏) | 侵入现实的解谜叙事 | 京都ARG、美术馆无限流解谜 |
| **城市游戏** | 以城市为游戏板的竞技 | Warp and Weft、冲进黑夜尽头 |
| **街头游戏** | 利用公共空间的即兴规则 | 高铁暗杀作家游戏 |
| **怪体育** | 规则异常的体育竞技 | VJ大战（"像电竞"） |

### 设计哲学：管杀不管埋

重轻在对话中最精辟的总结：

> "电子游戏建那个魔法圈它很强，你进去以后你除非点退出你才能退出来。但是我们建的这个圈就它没有电子游戏那种所有的都是给你搭建好的，它所有里面东西好多都需要从自己日常经验中去搭。" —— Tianqi

> "你们利用了现实世界唾手可得的白给的自由度。你建了一个门，你只有一个open，你摁F，打开这个门，或者关上这个门。是用一种没有希望的很穷举的方式……但是你看你们刚才说那个往俱乐部里边混也好，或者是去暗杀别人也好，你给那个目的是完全管杀不管埋，那个目的里并不包含手段。就是一个单不愣的目的，就我就扔给你，反正你自己看着办吧。然后他就可以 leverage 整个生活世界的一切的行动。" —— 重轻

**核心设计原则**：

| 原则 | 含义 | 反例 |
|------|------|------|
| **目标清晰，手段自由** | 只给"要做什么"，不给"怎么做" | 电子游戏：按 F 开门 |
| **现实即游戏引擎** | 街上的栏杆、路人、电梯、微信群都是游戏资源 | 密室逃脱：所有东西都是预设的 |
| **规则简单，涌现复杂** | 简单规则 + 真实环境 → 不可预测的行为 | RPG：所有可能性都被脚本覆盖 |
| **真实现场不可替代** | 游戏必须发生在"真的"演出/团建/派对里 | 模拟环境 |

## 五、代表作品深度解析

### 5.1 Warp and Weft（经与纬）

| 维度 | 内容 |
|------|------|
| **类型** | 街头竞技游戏 |
| **规则** | 两组人各一根16米绳子，60秒内利用街上一切东西围出最多边形 |
| **可用的"顶点"** | 手、路人、栏杆、汽车、电线杆、垃圾桶、石头、植物……一切 |
| **设计洞察** | 同一个游戏在商业街 vs 空旷地，策略完全不同；每走20米就是一个新游戏 |
| **哲学意义** | 用一根绳子重新"阅读"城市空间——栏杆不再是栏杆，是顶点；路人不再是路人，是队友 |

> "这个游戏它是你每去一个新的地方，或是你再走 20 米，可能就完全又是一个不同的解法。"

**它代表的设计方法**：最小干预（一根绳子）、最大涌现（无限种围法）、城市即游戏板。

### 5.2 冲进黑夜尽头

| 维度 | 内容 |
|------|------|
| **类型** | 百人城市追逐游戏（改编自 SF0 的 Journey to the End of the Night） |
| **规模** | 100+ 玩家，3小时 |
| **规则** | 3个"僵尸"追捕 → 被抓者变为僵尸继续追 → 幸存者需跑完5个检查点 |
| **路线** | 新天地 → TS淮海 → K11 → 集社 → 终点 |
| **checkpoint 设计** | 手机电量必须为7的倍数、在街上找追捕者远远合影 |
| **结果** | 100+人出发，十余人到达终点，完美平衡 |
| **涌现行为** | 结盟与背叛（自发的）、谈判（"大哥我给你表演个才艺放我过去"）、躲在草丛里、研究建筑的员工通道 |
| **玩家反馈** | "他生活在这片区域，这个街区对他很熟，但他玩了这个游戏，觉得这个街道又很陌生" |

**关键设计机制**：

1. **僵尸传染**：被抓 ≠ 游戏结束，而是身份转换 → 100% 全程参与
2. **分散物料点**：为避免聚集引起注意，10 份物料分散在不同点、领完即消失
3. **代号系统**：玩家自编代号，领物料时报代号——"在商场羞耻的地方说我是沙漠之鹰"→ 即刻进入角色状态
4. **微信群作为游戏层**：群聊成为第二游戏空间，玩家在群里制造叙事
5. **本土化 (localize)**：原作是旧金山的游戏，在上海需要重新选点、考虑"哪里会被保安抓"

> "就是感觉小学三年级以后我就再没有体验过的东西，就是在操场上一堆傻小孩在临时发明规则。" —— 重轻

### 5.3 Club with No Entry（没有门的俱乐部）

| 维度 | 内容 |
|------|------|
| **类型** | LARP / 社会实验 |
| **场地** | SYSTEM 俱乐部，Joanna 当晚在此真实演出（VJ） |
| **规则** | 演出前 40 分钟，玩家必须**不买票**混入俱乐部 |
| **设计动机** | 对俱乐部"势利文化"的 subvert——VIP 名单、开卡、门口的歧视性筛选 |
| **涌现的高光** | 真外卖员提着外卖袋戴着 headlamp 冲进来（所有人都以为是玩家，结果是真送外卖的） |

**深层意义**：

> "我很喜欢观察一些本身特别严肃，或者是有很多 hidden rules 的地方，尤其是有很多那种 hidden 的 power structure。" —— Joanna

这个游戏不是关于"逃票"，而是关于**让隐形的权力结构变得可见**。俱乐部门口的歧视、酒桌的 Hierarchy、消费主义的势利——通过游戏规则把这些平时"正常"的东西变得荒诞。

> "真外卖员……就是那个外卖员如果没有这个设计的话，他就是这个城市生活里最乏味的一个东西……但是他却就是让人有那种 dramatic 的 experience。" —— 重轻

**外卖员的意外闯入是整个修四边形实践的完美隐喻**：现实本身比任何设计都更丰富，设计师的工作不是控制现实，而是创造一个框架让现实的戏剧性自然涌现。

### 5.4 办公室暗杀游戏

| 维度 | 内容 |
|------|------|
| **类型** | 一周长线渗透游戏 |
| **场地** | 某游戏公司的真实办公室 + 工作流 |
| **人数** | 100+ 员工（含老板） |
| **规则** | 暗黑三角人格测试 → 排战力值 → 分配暗杀目标（100人连成一个圈）→ 多种暗杀方式 |
| **暗杀方式** | 电梯里脚对脚5秒、发红包对方收了=死、发红包对方拒收=死、让目标在合同里偷偷加上"修四边形"三个字再发回来 |
| **社会动力学** | 形成派系 → 老板有自己的"护卫队" → 有人请年假只为不死 → 跨楼暗杀原本不交流的同事 |
| **设计目标** | 服务于公司"让不同组的人互相认识"的团建目标 |

**设计方法论洞察**：

- 游戏规则**嫁接**在真实工作流上（微信、OA、合同、电梯、工位）
- 暗黑三角测试不是心理评估而是**游戏数值设计**（"战力值"）
- "100人连成一个圈，我杀你、你杀他"——最简单的拓扑结构产生最复杂的社会动力学
- 最终结果远超设计预期："有些都他们自己发明的"

### 5.5 VJ 大战

| 维度 | 内容 |
|------|------|
| **类型** | 竞技赛事 |
| **规则** | 12组选手，只能用浏览器播放已存在的在线视频，配合 DJ 音乐进行 VJ battle |
| **观众** | 现场投票 |
| **技术手段** | AppleScript 控制窗口、浏览器插件、TouchDesigner、MIDI 控制器、纯 YouTube 快进键 |
| **核心评判标准** | 不是技术能力，而是 **curation**——选什么片、怎么配音乐 |

> "乱拳打死老师傅……这个活动特别 amazing 的一个点，是从第一秒钟开始，这个 VJ 的正统性就是被抛到窗外。" —— 重轻

> "这个人他是否上网？是否玩游戏？他是否是真的在这个环境里？……是否是真的热爱互联网？" —— Joanna

**深层意义**：VJ 大战是修四边形对"专业话语"的又一次 subvert。不是否定技术，而是把技术拉回它本来的位置——**手段而非目的**。真正被奖励的不是"最会玩 TouchDesigner 的人"，而是"看了最多 YouTube 视频、对互联网文化最敏感的人"。

## 六、设计方法论：从播客中提取的 8 条原则

### 原则 1：真实现场不可替代

> "如果我们再做一个原创密室……肯定是一个目标。但主要是这种比较好玩一点。因为演出得是真演出，舞会是真舞会，聚会是真聚会。" —— Joanna

修四边形不建造模拟环境。他们的游戏寄生在已经存在的真实事件上——真的俱乐部演出、真的公司团建、真的高铁旅程。这降低了搭建成本，同时让游戏的"真实性"远远超过任何模拟。

### 原则 2：message 是涌现的，不是预设的

> "我自己不管做创作我是很难一开始想到说我想表达啥……但是这个东西成为一个作品也好，大家去玩这个东西他大概会有 emerge 什么样的一个情感或者是思考，其实很多是我们就是没有 expectation 的。" —— Joanna

与"观念艺术"不同，修四边形不从 message 出发。他们搭一个舞台，让玩家自己在上面产生情感和思考。这不是放弃表达，而是**把表达的权利部分让渡给玩家和现实本身**。

### 原则 3：Design 是骨架，涌现是血肉

> "我觉得比如说我们做东西跟一些 Art Gallery、一些特别行为的一个区别，还是我们是有很多 Design 在里面的，它不仅仅是一个 Expression。" —— Joanna

修四边形不是"把一群人在街上撒出去就完了"。每个游戏都有经过设计的规则骨架——checkpoint 数量、传染机制、战力值排位、物料分散策略。Design 保证游戏"能玩完"，涌现保证游戏"超乎想象"。

### 原则 4：规则是媒介，不是包装

> "我是觉得他没有利用游戏这个媒介……他没有让玩家通过接受、进入这个规则而感受到作者的表达。就他没有把表达放到这个规则建立里。" —— Tianqi

Tianqi 在此提出了修四边形的**媒介伦理**：游戏化（gamification）的失败在于"套皮"——给大富翁套一个环保主题。真正的游戏设计是把表达放在规则里——玩家在遵守规则的过程中，自然体会到设计者的意图。

**例子**：办公室暗杀游戏中，"你必须让同事在不经意间把'修四边形'三个字写进合同再发回给你"——这个规则本身就是对职场沟通荒诞性的表达。不需要额外解释。

### 原则 5：Playtest 是游戏媒介的独特优势

> "游戏测试会辅助这个创作者去修正他使用媒介的方式……它不是一个纯表达。" —— Tianqi

> "好多艺术挺难有反馈的，除了自我反馈。" —— Tianqi

Playtest 让游戏设计区别于其他艺术形式：你可以**直接看到人们怎么玩**，然后修正。这是一种**经验主义的设计方法**——不靠理论推导，而靠真实的玩家行为反馈。

### 原则 6：production cycle 要短

> "这个 format 它比电子游戏的好处当然就是 production cycle 很短。电子游戏经常，你也许追寻的是同样的东西。但它太累了，太贵了，太多人了。" —— Joanna

修四边形选择 pervasive game 不仅是哲学选择，也是**实践智慧**：电子游戏的制作周期（动辄数年）使得创作者必须在没有真实反馈的情况下"盲信"自己的方向。短周期 = 更多迭代 = 更接近真实。

### 原则 7：灵光一闪是可以训练的手艺

> "我不觉得灵光一闪它不是个手艺。你闪多了以后，你会知道哪个更闪。看过更亮的东西你就知道其他的东西没那么亮。所以那个它会形成一种标准，就是你会越来越快的觉得什么东西是好的。" —— Tianqi

> "尤其是做现实中的一些东西，更多是说想象它发生的时候是什么样子的？你这个想象是否跟现实真正去发生的时候是更接近的？所以你想的越准，可能它成所谓成功率就会越高。" —— Joanna

两人给出了互补的答案：Tianqi 强调**品味**（看过好的才知道什么是好的），Joanna 强调**预判**（想象现场越准，成功概率越高）。

### 原则 8：保持与世界的联系

> "我觉得一个人的意义就是他跟这个世界的联系……游戏设计它很难不去想象玩家，你很难不去跟真实世界和这些玩家有联系……它都会让你觉得跟这世界有联系。" —— Tianqi

这是整场对话最深层的底色。Tianqi 把游戏设计描述为**一种与世界保持联系的方式**——不是自说自话的自我表达，而是持续地想象他人、接受他人反馈、被他人的行为所改变。

## 七、社群基础设施：茶水间

"茶水间"是修四边形运营的定期游戏创作者聚会——没有在播客中被详细展开，但从上下文可以拼出：

- **形式**：定期 meetup，创作者分享"半成品"
- **规模**：从"不小心来了一百多人"到稳定运行
- **精神**：分享过程而非成品，降低参与门槛
- **作用**：连接上海独立游戏/创意技术/艺术亚文化圈层

**茶水间之于修四边形，类似于市集之于共益企业**——它是社群的基础设施，是创作者彼此发现的空间，也是修四边形从"做游戏的组织"变成"让别人的游戏发生的平台"的关键一步。

---

## 八、与 B Community 的深层共振

修四边形的实践与整个 B Community 分析框架高度共振：

| 维度 | 共振点 | 证据 |
|------|--------|------|
| **价值观** | 人先于利润、真实先于表演、自由先于安全 | 离开大厂、免费活动、不受 gallery 绑架 |
| **方法论** | 工作坊、市集、沙龙 | 茶水间、HTML Day、Game Jam |
| **经济模式** | 交叉补贴 | 商业项目 (NIKE等) → 养独立创作 |
| **叙事弧线** | 发现问题→拒绝主流→直接建造 | 完整的五步弧线 |
| **空间类型** | 街道、俱乐部、办公室 | "非传统空间"作为游戏场 |
| **时间伦理** | 反加速 | production cycle 短，不求规模 |

**修四边形的独特贡献**：它展示了"游戏"作为 B Community 中一种尚未被充分命名的方法论——不是 gamification（套皮游戏化），而是 game as social fabric（游戏作为社会连接的织造术）。

---

## 九、修四边形的边界与未完成

### 他们不做什么

- 不做纯电子游戏（尽管成员有电子游戏背景）
- 不做被 gallery 绑架的"当代艺术"（尽管在美术馆做过项目）
- 不做 gamification 式的"套皮游戏"
- 不做大规模复制（每一个游戏都是 site-specific）

### 他们想要但还没做的

- **SCP Foundation 式的长青项目**：有线上部分也有线下部分的持续叙事宇宙
- **公共交通系列**的深化：高铁暗杀作家游戏的系列化
- **Viral 项目**：虽然"不会以 viral 为目的去设计"
- 不做中国的 Mr. Beast（"他的游戏其实设计得很差"）

### 内在张力

1. **一期一会 vs 可持续**：大部分游戏是"今晚很快乐，第二天就没了"——如何积累？
2. **上海 local vs 更大范围**：所有项目深度依赖上海的城市肌理——能否移植？
3. **好玩 vs message**：Joanna 和 Tianqi 对"是否需要有 message"有微妙差异
4. **边缘 vs 主流**：拒绝被主流收编，但 MSCHF 式的"toC 但不弯腰"是理想

---

## 十、总结：修四边形是什么

**修四边形是一个用游戏修补现实的集体。**

- "游戏"不是产品而是**媒介**——规则是语言，playtest 是编辑，玩家的自发行为是正文
- "修补"不是重建而是**撬开**——在四边形的缝隙中插入一个短暂的魔法圈
- "现实"不是背景而是**引擎**——街道、电梯、微信、合同、俱乐部，一切皆可为游戏资源
- "集体"不是公司而是**乐队**——四个人，没有长期规划，被"好玩"的标准凝聚在一起

他们在做的事情，用重轻的总结来说：

> "人们会享受，而且你是被人们需要的，它是有一些创造一些很独特的、难忘的留在人心里的一些东西。"

**在 B Community 的共益星图中，修四边形（以阿久和天琦为核心）代表一种独特的行动范式：不提供服务、不制造产品、不倡导政策——而是在现实世界的裂缝中创造短暂的"另一个世界"，让人在其中重新学会玩、重新看见彼此、重新感知自己生活的城市。**
