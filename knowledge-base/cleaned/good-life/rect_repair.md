# 阿久 · 按良好生活标准分类

> **良好生活三问**：
> 1. 是否仍然愿意想象、表达、创造——在既有路径之外，把新的可能带进现实？
> 2. 是否在不确定的现实里，诚实地面对自己，找到内在自洽与韧性？
> 3. 是否在关照自我的同时，关切他人、回应周遭、参与塑造共同世界？

---

## 一、创造力 / Create Well

> *"仍然愿意想象、表达、创造，是在既有路径之外，在线性的成功学叙事之外，并把一些新的可能带进现实。"*

### 1.1 拒绝既有路径：从电子游戏出走

阿久的创造力起点是**系统性的拒绝**。在互娱大厂做电子游戏时，她看到的不只是个人的职业倦怠，而是整个媒介范式的困境：

> "它太赚钱了，被太多东西绑架了。"

这不是"做一个更好的电子游戏"的改良冲动，而是对**"电子游戏=游戏"这一单一叙事的彻底质疑**。她选择离开的不仅是公司，更是整个被资本绑架的电子游戏工业。

| 拒绝对象 | 具体内容 |
|----------|----------|
| 大厂职业路径 | 2023年主动离开互娱大厂，放弃高薪稳定 |
| 电子游戏媒介 | 不选择"做更好的电子游戏"，而是直接离开这个媒介 |
| 线性成功学 | 不追求 VC、不追求规模、不追求快速变现 |
| 当代艺术体制 | 不被 gallery 绑架，不做"观念艺术"式的自上而下表达 |

### 1.2 发现新可能：Pervasive Game 的转向

**转折点**：2024年参加雅典 Trust in Play camp（European School of Game Design）。

> "发现其实游戏在很多其他领域，它其实可以再跟很多东西进行交互的。然后我可能发现，诶，玩还是特别好玩，就 play 还是一个特别好的东西。"

她把"玩"从电子屏幕中解放出来，放进真实世界——街道、俱乐部、办公室、高铁、美术馆。这是一种**媒介层面的创造**：不是改良旧媒介，而是发明新的游戏发生方式。

**理论根基**：阿久引用了学术著作《Pervasive Games》，将实践放在一个更广阔的谱系中：

| 子类型 | 说明 | 修四边形案例 |
|--------|------|-------------|
| LARP（实况角色扮演） | 参与者在现实空间中扮演角色 | Club with no entry、Dancefloor Assassination |
| ARG（另类现实游戏） | 侵入现实的解谜叙事 | 京都ARG、美术馆无限流解谜 |
| 城市游戏 | 以城市为游戏板的竞技 | Warp and Weft、冲进黑夜尽头 |
| 街头游戏 | 利用公共空间的即兴规则 | 高铁暗杀作家游戏 |
| 怪体育 | 规则异常的体育竞技 | VJ大战 |

### 1.3 代表作品：把新可能带进现实

| 作品 | 类型 | 年份 | 创造力的体现 |
|------|------|------|-------------|
| **Warp and Weft** | 街头竞技 | 2024 | 只用一根16米绳子，让城市的所有物体成为游戏资源——栏杆=顶点，路人=队友。最小干预，最大涌现 |
| **冲进黑夜尽头** | 百人城市追逐 | 2024.10 | 100人在上海夜晚从新天地逃到终点，僵尸传染机制。让熟悉的街区变得陌生 |
| **Dancefloor Assassination** | LARP | 2024 | 在真实舞池里发生的暗杀游戏，把俱乐部变成剧场 |
| **第一届网络视频VJ大战** | 线上+线下竞技 | 2025.03 | 只能用互联网已有的视频做VJ素材——把"curation"变成竞技，把技术正统抛到窗外 |
| **办公室暗杀游戏** | 一周长线渗透 | — | 嫁接在真实工作流上（微信、OA、合同、电梯），让100+员工（含老板）被游戏规则重新组织 |
| **HTML Day 2026** | 线下聚会 | 2026 | 在上海一起写网页——技术作为一种社交媒介 |

### 1.4 设计方法论：八条创造原则

从播客和实践中，可以提取出阿久与修四边形的创造方法论：

| # | 原则 | 说明 |
|---|------|------|
| 1 | **真实现场不可替代** | 游戏寄生在真实事件上——真演出、真团建、真派对。不建造模拟环境 |
| 2 | **message 是涌现的，不是预设的** | 不从"想表达什么"出发，而是搭舞台让玩家自己产生情感和思考 |
| 3 | **Design 是骨架，涌现是血肉** | 规则保证游戏"能玩完"，涌现保证游戏"超乎想象" |
| 4 | **规则是媒介，不是包装** | 把表达放在规则里——玩家在遵守规则的过程中，自然体会到设计者的意图 |
| 5 | **Playtest 是独特优势** | 直接看到人们怎么玩然后修正——经验主义的设计方法 |
| 6 | **production cycle 要短** | 电子游戏动辄数年 = 盲信；短周期 = 更多迭代 = 更接近真实 |
| 7 | **灵光一闪是可以训练的手艺** | 品味（看过好的） + 预判（想象现场越准，成功率越高） |
| 8 | **管杀不管埋** | 只给"要做什么"，不给"怎么做"——让玩家 leverage 整个生活世界的行动能力 |

### 1.5 命名即创造："修"而非"造"

> "修不了四边形，没有这个能力重建一个现实世界。" —— Tianqi
> "修这个字比较有美感吧。" —— Joanna

**修四边形**的名字本身就是一种创造性姿态：不是带着蓝图去建造一个新世界（那是技术乌托邦的傲慢），而是用手头已有的材料（上海的街道、商场、俱乐部、办公室、微信群）去**撬动和变形**。这种姿态同时是谦逊的（"造不了"）和激进的（"撬开"）。

## 二、身心安顿 / Live Well

> *"如何在不确定、不断变化、也并不总是友好的现实里，依然能够诚实地面对自己，慢慢找到一种内在自洽，同时保有韧性的生活方式。"*

### 2.1 诚实面对：承认"电子阳痿"

阿久的故事起点并非英雄式的觉醒，而是一种**诚实的疲惫**：

> "当时不是刚从大厂离职，觉得电子游戏已经没有什么特别，就觉得特别绝望，又加上这个电子阳痿，又加上被大厂折磨，觉得电子游戏没啥意思。"

"电子阳痿"是一个极其诚实的自我诊断——不美化、不回避、不把职业危机包装成英雄叙事。她承认自己热爱的媒介让她失望了，承认自己被大厂消耗了，承认自己不知道该往哪里去。

**这种诚实是"身心安顿"的前提**：如果不承认自己在哪里受了伤，就不可能找到治愈的路。

### 2.2 内在自洽：交叉补贴的经济伦理

阿久没有选择"要么赚钱、要么纯粹"的二元对立，而是找到了一条**交叉补贴**的中间道路：

| 维度 | 内容 |
|------|------|
| 主要收入来源 | 商业设计项目（NIKE等）、贴纸/merch销售、活动门票 |
| 免费/低利润活动 | 茶水间聚会、街头游戏、Game Jam——全部免费 |
| 经济逻辑 | 商业项目养独立创作——"够做下一件事"即可 |
| 拒绝的资本 | 大厂高薪、VC投资 |
| 利润率定位 | 不追求利润最大化，追求**可持续的创作自由** |

这种经济模式的内在自洽在于：**她知道自己需要多少钱、为什么需要这些钱、以及她不愿意为钱付出什么代价**。商业项目不是"妥协"，而是让独立创作得以持续的条件；独立创作不是"逃避"，而是商业技能被用在真正相信的事情上。

**参考对象**：阿久明确提到纽约创意集体 **MSCHF**——"又做有 message 的艺术项目，同时还是 toC 的，不需要被当代艺术绑架……同时也卖包。"这是一种在经济上自洽、在创作上不弯腰的姿态。

### 2.3 韧性：反加速的时间伦理

从2023年离开大厂到2026年，3年时间——没有追求VC、规模或快速变现，而是逐步积累：

```
2023 ──┬── 离开互娱大厂
       │
2024 ──┼── Trust in Play camp (雅典) ★ 转折点
       │
       ├── Warp and Weft 诞生
       │
       ├── 10月 冲进黑夜尽头 · 100人 ★
       │
2025 ──┼── 3月 第一届网络视频VJ大战
       │
       ├── 7月 茶水间稳定运行
       │
       ├── 京都ARG
       │
       ├── 上海↔纽约实时传真机
       │
2026 ──┼── 3月 全球 Game Jam
       │
       └── 4月 VTuber 阿久aku 引退 ★ 身份转换
```

**时间伦理特征**：
- **项目制而非增长制**：不是"越做越大"，而是"做完一个再做一个"
- **容忍不确定性**：茶水间从"不小心来了一百多人"到稳定运行——没有预设目标，允许自然生长
- **身份流动性**：从大厂设计师→媒体艺术家→游戏设计师→VTuber→社群组织者——不把自己锁死在单一身份里

### 2.4 身份转换：VTuber 引退的自我诚实

2026年4月，阿久引退了自己的 VTuber 身份"阿久aku"。这不是失败，而是**知道自己什么阶段需要什么身份**的自我觉知。修四边形持续运作，但 VTuber 身份已完成其使命——一种干净的、不需要解释的告别。

### 2.5 价值观作为内在锚点

在15个案例的价值观热力图中，阿久的五个核心价值观均达到高强度：

| 价值观 | 强度 | 如何在生活中体现 |
|--------|:---:|------|
| 人先于利润/数字 | ████ | 离开大厂、免费活动、拒绝"更赚钱"的路径 |
| 真实先于表演 | █████ | 茶水间分享"半成品"、面对面街头游戏、不包装 |
| 自由先于安全 | █████ | 离职、实验性创作、不确定的社群运营 |
| 边缘先于中心 | ████ | 在"街道"而非"美术馆"做游戏 |
| 关系先于交易 | █████ | 免费茶水间、game as social fabric |

**这些价值观不是宣言，而是每天在做的事情**——当阿久选择免费开放茶水间、选择在街头而非美术馆做游戏、选择分享"不完美的半成品"时，她在用行动而非言语兑现这些价值。

## 三、公共性 / Benefit Well

> *"如何在关照自我的同时，关切他人、回应周遭、关心公共，参与塑造我们共同身处的世界。"*

### 3.1 茶水间：为创作者建造的基础设施

茶水间是阿久和修四边形运营的定期游戏创作者聚会——它不是一个"项目"，而是一个**平台**：

| 维度 | 内容 |
|------|------|
| 形式 | 定期 meetup，创作者分享"半成品" |
| 规模 | 从"不小心来了一百多人"（小红书256赞）到稳定运行 |
| 精神 | 分享**过程而非成品**，降低参与门槛 |
| 门槛 | 不是展览/讲座——不用"准备好"才能来，带着半成品就可以 |
| 作用 | 连接上海独立游戏/创意技术/艺术亚文化圈层 |
| 费用 | 免费 |

> "想让创作者们来讲自己最近在做的东西。"

**茶水间的公共性意义**：它让上海的独立游戏创作者有了一个可以彼此看见、彼此连接的空间。在商业游戏工业之外，在当代艺术体制之外，在学术圈之外——一个属于创作者自己的"第三空间"。它是**让别人的创造得以发生**的基础设施。

### 3.2 街头游戏：让陌生人重新连接

阿久的 pervasive game 本质上是**公共空间的重新激活**：

**Warp and Weft**：两个陌生人因为一根绳子被迫协作。栏杆不再是栏杆，是顶点；路人不再是路人，是队友。**游戏让人重新"阅读"城市。**

**冲进黑夜尽头**：100人在上海夜晚奔跑，僵尸传染机制让失败者变成追捕者——没有"出局"，只有身份的转换。玩家反馈说："他生活在这片区域，这个街区对他很熟，但他玩了这个游戏，觉得这个街道又很陌生。"**游戏让人重新感知自己生活的城市。**

**Club with No Entry**：让玩家不买票混入俱乐部——不是教人逃票，而是**让隐形的权力结构（VIP名单、门口歧视、消费主义势利）变得可见**。真外卖员的意外闯入成为整个实践的完美隐喻：现实比设计更丰富，设计师要做的不是控制，而是创造让现实戏剧性自然涌现的框架。

### 3.3 关系重建的三个层次

阿久的实践在不同尺度上重建了人与人的连接：

```
层次 1：玩家与玩家之间
└── 街头游戏中陌生人被迫协作、结盟与背叛、谈判
    "大哥我给你表演个才艺放我过去"

层次 2：创作者与创作者之间
└── 茶水间去中心化的 peer community
    不是展览/讲座，而是分享半成品 → 降低门槛 → 彼此看见

层次 3：人与城市之间
└── 通过游戏重新感知和使用公共空间
    街道不再是通道，而是游戏板；城市不再是背景，而是引擎
```

### 3.4 国际连接：游戏作为世界语言

阿久的实践不是封闭的本地社群，而是有意识地连接国际：

| 连接 | 方向 | 内容 |
|------|------|------|
| 雅典 → 上海 | 输入 | Trust in Play camp 带来 pervasive game 的视野 |
| 上海 → 纽约 | 输出 | 实时传真机、游戏交换 |
| 全球 Game Jam | 双向 | 2026年3月，连接国际独立游戏社群 |
| MSCHF | 参照 | 纽约创意集体的模式作为修四边形的参考 |

### 3.5 边界与选择：什么不是公共性

阿久的公共性不是无差别的。她明确不做什么，这些"不做"恰恰定义了公共性的边界：

| 不做的事 | 原因 |
|----------|------|
| 不做纯电子游戏 | 电子游戏"太赚钱了，被太多东西绑架了" |
| 不做被 gallery 绑架的"当代艺术" | 尽管在美术馆做过项目，但不被体制收编 |
| 不做 gamification 式的"套皮游戏" | "没有利用游戏这个媒介"——规则是媒介，不是包装 |
| 不做大规模复制 | 每一个游戏都是 site-specific，依赖具体的城市肌理 |
| 不做中国的 Mr. Beast | "他的游戏其实设计得很差" |

### 3.6 独有贡献：将"游戏"确立为公共行动的方法论

在 B Community 的所有案例中，阿久/修四边形贡献了一种尚未被充分命名的方法论：

> **Game as social fabric（游戏作为社会连接的织造术）**

不是 gamification（套皮游戏化）——给大富翁套一个环保主题；而是让**规则本身成为媒介**，玩家在遵守规则的过程中自然体会到设计者的意图。办公室暗杀游戏中"必须让同事在合同里偷偷加上'修四边形'三个字"——这个规则本身就是对职场沟通荒诞性的表达。

**Tianqi 的总结**：
> "游戏设计它很难不去想象玩家，你很难不去跟真实世界和这些玩家有联系……它都会让你觉得跟这世界有联系。"

**这不是关于"做游戏"，而是关于通过游戏保持与世界的联系。**
