# 赵伊人 · 按良好生活标准分类

> **良好生活三问**：
> 1. 是否仍然愿意想象、表达、创造——在既有路径之外，把新的可能带进现实？
> 2. 是否在不确定的现实里，诚实地面对自己，找到内在自洽与韧性？
> 3. 是否在关照自我的同时，关切他人、回应周遭、参与塑造共同世界？

---

## 一、创造力 / Create Well

> *"仍然愿意想象、表达、创造，是在既有路径之外，在线性的成功学叙事之外，并把一些新的可能带进现实。"*

### 1.1 游戏作为社会分析工具："浮萍定海"的方法论发明

赵伊人的创造力体现在一种**媒介跨界的能力**——将 LARP（实况角色扮演游戏）从娱乐场域迁移到社会议题的讨论空间中。与赵蒙旸、殷艾雯共同创作的"浮萍定海"（Floating Duckweed Settling Sea）是一个大规模 LARP 游戏，其核心创造在于：**用游戏机制让隐形的照护劳动变得可见、可量化。**

这不是一个"关于照护劳动"的讲座或展览，而是一个让参与者**在游戏过程中亲身经历照护经济学逻辑**的体验装置。游戏的规则本身即是媒介——玩家在遵守规则、做出选择的过程中，自然体会到照护劳动在社会中被系统性地低估和隐形的困境。

| 创造维度 | 具体内容 |
|----------|----------|
| 媒介创新 | LARP 作为社会分析工具——不用于娱乐或奇幻叙事，而用于呈现社会现实的结构性问题 |
| 知识生产 | 让照护劳动从"抽象概念"变成"可量化的游戏机制"——这是一种新的知识传递方式 |
| 合作创造 | 与殷艾雯、赵蒙旸共同创作——三位来自不同背景的实践者，在交叉点上生成新可能 |
| 可体验的理论 | 不是"告诉你"照护劳动被低估，而是"让你在游戏中体验到"它——身体知识先于抽象认知 |

### 1.2 发现新可能：定海桥互助社作为"另类机构"

定海桥互助社本身就是一种**制度层面的创造**——它不是政府的社会服务机构，不是 NGO，不是商业社区中心，而是一种**自组织、互助为基础的另类机构**。

赵伊人作为定海桥互助社的长期参与者（自约2015年起），深度参与了这一另类机构的构建与运营。在上海市杨浦区定海桥——一个历史上是工人阶级聚居区、如今正经历城市更新改造的地段——定海桥互助社创造了一个**不属于任何既有系统**的空间：

| 另类机构的特征 | 具体内容 |
|----------|----------|
| 自组织形式 | 非政府、非基金会、非商业——由参与者共同维护和决策 |
| 互助逻辑 | 不是"服务提供者—受益者"的慈善模式，而是平等的互助关系 |
| 空间在地性 | 扎根定海桥——不脱离具体的地理和社区脉络 |
| 反制度化弹性 | 不追求成为"正式机构"，保持小规模、可调整、有生命力的状态 |

### 1.3 代表实践：将艺术、教育与社区组织融为一体

| 实践 | 领域 | 创造力体现 |
|------|------|-------------|
| **"浮萍定海"LARP** | 游戏/社会介入 | 用 LARP 游戏机制呈现照护经济学——游戏从娱乐媒介变为社会分析工具。与殷艾雯、赵蒙旸合作创作 |
| **定海桥互助社** | 社区组织/另类机构 | 在上海边缘地带长期运营一个自组织的互助空间——制度创造 |
| **"流体愈学"** | 艺术/教育/疗愈 | 将治愈与学习转化为公共实践——打破"治疗是私人的、学习是制度化的"边界 |
| **自我教育实践** | 教育 | 不以"教—学"的等级关系组织，而是以共同探索的方式——教育作为一种平等的共同生活 |

### 1.4 创造方法论：跨界的编织术

赵伊人的创造方法论不是"在某个领域内精进"，而是**在不同领域之间编织连接**：

| # | 原则 | 说明 |
|---|------|------|
| 1 | **游戏作为方法，不是目的** | LARP 不用于娱乐，而用于社会分析——媒介服务于问题，而非问题服务于媒介 |
| 2 | **合作即创造** | "浮萍定海"是与殷艾雯、赵蒙旸的合作产物——创造不是孤立的个人行为，而是在对话和碰撞中生成 |
| 3 | **让隐形变得可感知** | 照护劳动、社区关系、互助网络——这些"看不见"的东西，通过游戏机制被具身化 |
| 4 | **制度即作品** | 定海桥互助社的运营方式本身就是一种创造——不是"做一个关于互助的作品"，而是"做一个互助的机构" |
| 5 | **不急于被命名** | 实践在前，归类在后——不被"这是艺术""这是教育""这是社工"的边界束缚 |

### 1.5 命名即创造："浮萍定海"的诗学

"浮萍定海"这个名字本身就是一种创造性表达——浮萍（漂泊、无根、微小）与定海（安定、锚定、宏大）之间的张力，精确地捕捉了当代中国城市中个体生命的处境：微小如浮萍，却仍然在尝试为彼此创造一片可以定下来的海。这不是自上而下的宏大叙事，而是**以微小行动回应宏大困境**的诗学。

## 二、身心安顿 / Live Well

> *"如何在不确定、不断变化、也并不总是友好的现实里，依然能够诚实地面对自己，慢慢找到一种内在自洽，同时保有韧性的生活方式。"*

### 2.1 长期锚定：社区的持续在场

赵伊人的身心安顿之道在于**长期承诺**。自约2015年起深度参与定海桥互助社——这不是"做几个项目然后离开"的项目制逻辑，而是**将自己的一部分生命嵌入一个具体的社区和地点**。

这种长期性本身就是一种对当代加速主义的抵抗。在项目制（做三个月、汇报、拿下一笔钱）、流动制（哪里有资源去哪里）、曝光制（追求被看见、被认可）成为默认工作方式的时代，**持续十年地留在一个社区**是一种激进的选择。

| 维度 | 内容 |
|------|------|
| 时间尺度 | 约2015年至今，十年以上的持续参与 |
| 空间锚点 | 上海杨浦定海桥——一个具体的、有历史的、正在变化的街区 |
| 关系积累 | 不是每一次都"从零开始建立信任"，而是累积性的社区关系 |
| 内在节奏 | 不追求"快速产出"，容忍缓慢和反复 |

### 2.2 互助作为日常实践：不是意识形态，是生活本身

定海桥互助社的核心理念——"互助"——不是在纸面上讨论的意识形态，而是**每天在做的事情**：

> 互助不是"我帮你"，而是"我们一起"。

在定海桥互助社中，没有人是纯粹的"服务提供者"或纯粹的"受益者"——每个人既是给予者也是接受者。这种**非等级化的互助关系**提供了赵伊人内在自洽的伦理基础：不需要扮演"帮助者"的道德优越感，也不需要承受"被帮助者"的亏欠感。

### 2.3 在城市边缘安顿：定海桥作为锚点

定海桥是一个历史上工人阶级聚居的上海街区，正经历城市更新和再开发。选择在这里扎根——而不是在艺术区、高校周边或中产社区——本身就是一种**自我定位**：

- **与主流保持距离**：不在艺术圈的中心，不在知识圈的中心，不在商业圈的中心
- **与具体的人在一起**：不是抽象的"无产阶级"概念，而是定海桥的邻居们
- **接受变化**：定海桥本身在变化（改造、拆迁、人口流动），社区工作在流动中寻找锚点

### 2.4 自我教育作为生活方式

赵伊人将"自我教育"作为生活实践——不是去机构里被教育，也不是去教育别人，而是**和他人一起学习**。这种自我教育不是获取文凭或技能的手段，而是一种**对待世界的基本态度**：保持好奇、保持开放、保持学习的状态。

这与定海桥互助社的精神高度一致——互助社不仅是一个"做活动"的地方，更是一个**共同学习、共同成长**的共同体。

### 2.5 价值观作为内在锚点

从赵伊人的实践中，可以提取出以下核心价值观：

| 价值观 | 强度 | 如何在生活中体现 |
|--------|:---:|------|
| 互助先于竞争 | █████ | 互助社的基本逻辑——不是谁做得更好，而是一起做 |
| 长期先于速效 | ████ | 十年对一个社区的持续投入 |
| 过程先于成果 | ████ | 互助作为日常实践，而非为了产出"项目报告" |
| 边缘先于中心 | ████ | 扎根定海桥而非市中心、艺术区 |
| 平等先于等级 | █████ | 互助社的非等级化组织方式，自我教育的共同探索模式 |

## 三、公共性 / Benefit Well

> *"如何在关照自我的同时，关切他人、回应周遭、关心公共，参与塑造我们共同身处的世界。"*

### 3.1 让照护劳动可见："浮萍定海"的公共教育

赵伊人公共性实践最核心的贡献是**让系统性地被隐形的照护劳动变得可见**：

| 维度 | 内容 |
|------|------|
| 问题 | 照护劳动（照料老人、抚育儿童、情感劳动、社区维护）在社会经济体系中被系统性地低估、不计入GDP、不被视为"真正的工作" |
| 方法 | 用 LARP 游戏机制将其量化——玩家在游戏中做出的照护选择直接影响游戏进程和结果 |
| 效果 | 参与者不是"听到"照护劳动很重要，而是在游戏过程中**体验到**它的不可或缺 |
| 超越性 | 这不是一个"关于照护劳动"的作品，而是一个**让照护劳动被感知**的体验装置 |

**"浮萍定海"的公共性意义**：它让每一个参与游戏的人——这些人可能包括艺术从业者、学生、社区工作者、普通市民——亲身经历了一套**不同于主流经济学的价值衡量体系**。在这个体系里，照护不是附属品，不是"真正工作之外"的事，而是整个社会运转的基石。这种体验比任何讲座、文章、口号都更有力地改变了参与者对"什么是重要的工作"的理解。

### 3.2 社区自主：定海桥互助社作为公共基础设施

定海桥互助社的长期运营本身就是一项**公共基础设施的建设**——不是物理意义上的"盖楼"，而是社会意义上的"织网"：

```
层次 1：空间层
└── 在定海桥提供一个实体空间——可以聚集、讨论、创作、休息

层次 2：关系层
└── 社区邻里之间建立互助网络——不是你帮我我帮你，
    而是大家一起维护一个共同的生态

层次 3：制度层
└── 自组织、互助为基础的"另类机构"——证明了
    不依赖政府和资本的社区自治是可能的

层次 4：认知层
└── 让更多人看到：上海不只有陆家嘴和法租界，
    还有定海桥——还有另一种城市生活的可能
```

赵伊人在其中扮演的角色不是"领导者"或"创始人"，而是一个**长期的、持续的参与者和编织者**——不是建立机构然后离开，而是在日常中一针一线地编织社区关系。

### 3.3 连接体制与基层：艺术机构与社区的双向通道

赵伊人实践的一个独特之处在于**横跨体制与基层两个世界**：

| 面向 | 连接对象 | 内容 |
|------|------|------|
| 艺术体制 | 上海双年展、上海当代艺术博物馆（PSA） | 将定海桥互助社的实践带入艺术机构的讨论中——让"社区"不只是当代艺术的修辞 |
| 社区基层 | 定海桥居民、互助社参与者 | 将艺术机构的资源和注意力反向导入社区——不是"艺术家下乡"，而是双向的、平等的关系 |
| 学术界 | 赵蒙旸（合作者——学术背景） | 将学术理论（照护经济学、性别研究）落地为可体验的游戏机制 |

这种**横跨不同世界的能力**本身就是一种公共性——不是站在任何一个单一系统内部说话，而是在不同系统之间**翻译、连接、创造对话**。

### 3.4 "流体愈学"：治愈作为公共实践

"流体愈学"项目——赵伊人参与的另一项实践——将治愈和学习从私人领域拉入公共空间：

- **治愈不是私人的**：创伤和痛苦往往有社会根源——在公共空间中被看见、被讨论、被共同面对，本身就是治愈的一部分
- **学习不是制度化的**：不是去学校"被教育"，而是和同伴一起探索、一起成长
- **公共实践**：将通常被私人化或医疗化的"治愈"重新置于社会关系网络中

### 3.5 边界与选择：什么不是公共性

赵伊人的公共性同样有明确的边界：

| 不做的事 | 原因 |
|----------|------|
| 不做一次性"社区艺术项目" | 一次性的介入无法建立真正的信任和关系——需要长期在场 |
| 不把社区当作材料 | 定海桥互助社不是"艺术家的田野调查对象"，而是共同生活的空间 |
| 不追求体制化 | 不定海桥互助社成为"正式机构"——保持小规模、去中心化、灵活调整的能力 |
| 不做单向"赋能" | "我来赋能你们"是傲慢的——互助不是单向的给予 |
| 不让照护劳动继续隐形 | 整个"浮萍定海"的核心使命——打破这种系统性的隐形 |

### 3.6 独有贡献：让"照护"成为公共话语的核心

在 B Community 的所有案例中，赵伊人贡献了一种尚未被充分关注的公共性方法论：

> **照护劳动的可视化与公共化（Making Care Labor Visible and Public）**

这不是关于"做好事"或"帮助弱势群体"的慈善叙事，而是关于**重新定义什么是"有价值的工作"**。在一个将 GDP 增长、技术创新、资本回报视为唯一重要指标的社会中，赵伊人通过定海桥互助社的日常实践和"浮萍定海"的游戏机制，提出并实践了一个根本性的追问：

**如果照护彼此不再是被忽视的附属品，而是社会运转的核心——这个世界会是什么样子？**

这不是一个空泛的哲学问题，而是一个被游戏化的、可体验的、可参与的公共实践。
