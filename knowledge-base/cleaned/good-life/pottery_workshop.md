# 乐天陶社 · 按良好生活标准分类

> **良好生活三问**：
> 1. 是否仍然愿意想象、表达、创造——在既有路径之外，把新的可能带进现实？
> 2. 是否在不确定的现实里，诚实地面对自己，找到内在自洽与韧性？
> 3. 是否在关照自我的同时，关切他人、回应周遭、参与塑造共同世界？

---

## 一、创造力 / Create Well

> *"仍然愿意想象、表达、创造，是在既有路径之外，在线性的成功学叙事之外，并把一些新的可能带进现实。"*

### 1.1 拒绝既有路径：从香港到景德镇的逆向迁徙

乐天陶社的创造力起点是**地理与文化上的"反向流动"**。1985年成立于香港——一个以金融和贸易为核心的城市——却在2005年选择进入景德镇，一个当时被主流叙事视为"没落的传统手工业城市"。这不是降级，而是一种创造性的空间选择：**把当代陶艺的活力注入千年瓷都，让传统工艺与当代创意碰撞**。

| 拒绝对象 | 具体内容 |
|----------|----------|
| 香港的商业逻辑 | 不在香港做高端陶瓷生意，而是进入景德镇做社区建设 |
| 传统陶瓷的保守叙事 | 把景德镇从"仿古复制产地"重新定义为"原创陶瓷发源地" |
| 艺术家孤岛模式 | 不做孤立的艺术家工作室，而是建造一个完整的创意生态系统 |
| 纯商业市集模式 | 周六创意市集要求100%原创，拒绝批发转售 |

### 1.2 发现新可能：周六创意市集的"100%原创"规则

乐天陶社的周六创意市集不是一个普通的市集——它有一条不可妥协的规则：

> 100% original required — 每一件在售作品必须是创作者本人的原创。

这个规则的创造性在于：它把"原创"从一个模糊的理想变成了**一个可操作、可审查、可执行的门槛条件**。在景德镇——一个以仿古和拷贝闻名的城市——这条规则等于在说："我们不接受景德镇最擅长做的事。"

**"100%原创"规则的系统性效应**：

| 效应 | 内容 |
|------|------|
| 筛选效应 | 自动淘汰了批发商、仿制品贩子和非创作者 |
| 激励效应 | 年轻陶艺家知道"只有原创才能进场"——这迫使他们在创作上下真功夫 |
| 品牌效应 | 周六市集成为"可以买到真正的原创陶瓷"的代名词 |
| 溢出效应 | 原创的氛围带动了整个景德镇的创作生态转型 |

### 1.3 代表实践：国际艺术家驻场计划

乐天陶社的国际艺术家驻场计划是一个**双向创造系统**：

| 维度 | 内容 |
|------|------|
| 输入 | 国际陶艺家来到景德镇，接触千年制瓷工艺和材料 |
| 输出 | 本地陶艺家接触国际当代陶艺的观念和方法 |
| 基础设施 | 乐天提供工作室空间、窑炉、材料和技术支持 |
| 不可替代性 | 只有在景德镇才能获得"在地的陶瓷经验"——这在任何其他城市都做不到 |

这不是一个"请外国艺术家来办讲座"的单向知识传播，而是一个**让"景德镇经验"成为国际陶艺界不可替代的资源**的策略性创造。

### 1.4 创造"景漂"——一个无法被复制的文化现象

乐天陶社最大的创造不是任何一件陶瓷作品，而是 **"景漂"这个现象本身**。

| 维度 | 内容 |
|------|------|
| 定义 | 来自全国和全球的年轻陶艺创作者聚集在景德镇生活和创作 |
| 规模 | 已形成数以万计的"景漂"群体 |
| 催化剂 | 乐天陶社的周六市集、驻场计划和教育工作是景漂生态的核心节点 |
| 不可复制性 | 这个现象依赖于景德镇千年制瓷的物理基础 + 乐天陶社20年社区耕耘的结合 |

> 景漂 —— 一个改变了景德镇这座城市的文化现象。

**"创造一种现象"远比"创造一件作品"困难**——现象意味着无数人的自发参与、城市的空间重组、经济模式的转变和身份认同的重塑。乐天陶社没有"策划"景漂，但它创建了景漂的**基础设施**。

### 1.5 免费讲座与陶瓷教育：创造知识的公共性

乐天陶社长期提供**免费讲座和陶瓷教育**——这不是营销手段，而是一种创造性的知识分配方式：

| 维度 | 内容 |
|------|------|
| 形式 | 定期免费公开讲座 |
| 内容 | 陶瓷技术、当代陶艺观念、国际陶瓷趋势 |
| 受众 | 任何对陶瓷感兴趣的人——不需要报名、不需要交费、不需要"资格" |
| 效果 | 降低了陶瓷知识的获取门槛，让"想学"本身就是入场券 |

这种"免费"不是成本控制不了，而是**有意识地选择不把知识变成商品**。在知识付费盛行的时代，这是一种创造性的拒绝。

## 二、身心安顿 / Live Well

> *"如何在不确定、不断变化、也并不总是友好的现实里，依然能够诚实地面对自己，慢慢找到一种内在自洽，同时保有韧性的生活方式。"*

### 2.1 诚实面对：20年深耕一个地方

乐天陶社在景德镇的20年是一个**诚实的尺度**：

> 2005年进入景德镇，至今已超过20年。

在"风口"和"赛道"话语支配的商业文化中，20年深耕一个地方不是"高效"的策略。但正是这种"低效"的选择，让乐天陶社拥有了任何"风口企业"都无法拥有的东西：**在地性**。

| 维度 | 内容 |
|------|------|
| 时间投入 | 20年，不只是一个周期 |
| 空间选择 | 景德镇——不是上海、不是北京、不是杭州 |
| 诚实之处 | 承认"社区建设需要时间"——不假装可以2年"引爆"一个市场 |
| 不可替代性 | 20年的信任不能速成——手工艺人知道乐天不会跑 |

这种诚实不是被动的"没得选"，而是主动的"选了这个，就一直做下去"。

### 2.2 内在自洽：创始人个人投资2000万

乐天陶社的创始人郑祎**个人投资了2000万人民币**。这不是一笔"资本运作"的投资，而是一笔**从自己口袋里掏出来的、不需要向LP汇报的、可以亏掉的钱**。

| 维度 | 内容 |
|------|------|
| 投资来源 | 创始人个人资产 |
| 投资金额 | 约2000万人民币 |
| 投资方式 | 不是"融资"，是"投入"——不需要回报率承诺 |
| 投资去向 | 空间建设、设备购置、国际驻场计划、免费教育等 |
| 内在逻辑 | 如果这件事值得做，就不需要等别人来投资——自己先做 |

这种经济模式的内在自洽在于：**用自己的钱支持自己相信的事**。没有投资人需要被"退出策略"安抚，没有基金需要"18个月翻倍"。这意味着乐天陶社可以做一件任何VC投资的企业都不敢做的事：**不求回报的长期投入**。

### 2.3 韧性：在一个"没落"的城市做一件"过时"的事

2005年的景德镇在主流叙事中是一个"没落的传统手工业城市"——人口外流、经济不振、陶瓷产业被低价竞争挤压。进入景德镇不是"抓住风口"，更像是"逆风而行"。

**20年坚持的韧性来源**：

```
1985 ──┬── 成立于香港
       │
       ├── 在香港建立当代陶艺的声誉和网络
       │
2005 ──┼── 进入景德镇 ★ 关键选择
       │
       ├── 开始建立工作室、市集和社区
       │
2010 ──┼── 周六创意市集逐渐成熟，"景漂"现象开始出现
       │
       ├── 国际艺术家驻场计划扩大影响
       │
2015 ──┼── "景漂"成为全国性文化现象
       │
2020 ──┼── 免费讲座和陶瓷教育持续运作
       │
2026 ──┼── 20年深耕 ★ 从被质疑到被尊敬
       │
       └── 2000万个人投资——从未要求"退出"
```

**韧性的根基**：
- **不是"坚持信念"的抽象叙事，而是"钱是我自己的，时间是我自己的"的具体自由**
- 创始人没有上级、没有董事会、没有LP——这意味着不需要为短期业绩辩护
- 20年的积累产生了"你不可能被替代"的护城河——任何想复制乐天陶社的人必须再花20年

### 2.4 创始人的自我定位：让渡光环给社区

郑祎作为创始人有一个值得注意的特点：**她让乐天陶社比她自己更知名**。在媒体报道中，"景漂"和"周六市集"出现的频率远超创始人的个人故事。这不是无心之举，而是一种有意识的自我定位——**让社区成为主角，让自己成为背景**。

| 现象 | 含义 |
|------|------|
| 郑祎不强调个人IP | 不是做"郑祎的陶艺学校"，而是做"乐天陶社" |
| "景漂"而不是"乐天学员" | 命名权让渡给社区——他们不是"乐天的人"，是"景德镇的漂" |
| 免费讲座不以"大师"冠名 | 知识分享去个人化——讲座的价值在内容，不在头衔 |

这种自我定位本身就是"身心安顿"的体现：**不需要通过个人崇拜来确认自己的价值**。

### 2.5 免费作为内核：教育不应该是付费墙后面的东西

乐天陶社长期坚持免费讲座和免费教育。这在"知识付费"的时代是一个反常规的选择：

| 维度 | 内容 |
|------|------|
| 免费讲座 | 持续多年，对所有人开放 |
| 免费逻辑 | 陶瓷知识是公共遗产，不应该被变成"课程产品" |
| 代价 | 放弃了知识付费可能带来的收入 |
| 收获 | 降低了"景漂"的入门门槛——一个不富裕的年轻人也能接触到最好的陶艺知识 |
| 与使命的一致性 | 如果目标是"让更多人做原创陶瓷"，那么设置付费墙是逻辑矛盾 |

## 三、公共性 / Benefit Well

> *"如何在关照自我的同时，关切他人、回应周遭、关心公共，参与塑造我们共同身处的世界。"*

### 3.1 "景漂"现象：一个人改变一座城

乐天陶社对公共性的最大贡献不是任何一个具体的项目，而是**"景漂"这个现象**——一个足以改变城市命运的社群运动。

| 维度 | 内容 |
|------|------|
| 现象规模 | 数以万计的年轻创作者聚集在景德镇 |
| 城市改变 | 景德镇从"仿古复制产地"转变为"原创手工艺创新之都" |
| 催化机制 | 周六创意市集 + 国际驻场计划 + 免费教育 = 一个年轻创作者无法拒绝的生态 |
| 经济影响 | 陶瓷相关的新经济取代了低端仿制造——餐饮、民宿、材料供应链全面升级 |
| 文化影响 | "景漂"成为一个可识别的文化标签——景德镇不再只被"千年瓷都"的单一定义锚定 |

> 公共性是指一个人或一个组织，通过自己的行动，让一座城市的叙事被改写。

**乐天陶社没有"策划"景漂**。但它做了让景漂可以发生的事情：一个可以展示原创作品的市集、一个可以与国际交流的驻场计划、一个可以免费学习的教育体系。当这些条件同时存在时，年轻人自己选择了来——制造了"漂"。

### 3.2 周六创意市集：100%原创作为公共规则

周六创意市集的公共性不仅在于"让年轻人有地方卖东西"，更在于**它建立了一条公共规则，并且坚持执行**。

| 维度 | 内容 |
|------|------|
| 规则 | 100%原创作品，禁止批发转售 |
| 执行 | 不是"倡议"，是"不符合就别来"的门槛 |
| 公共效应 | 创造了一个"可以信任"的交易空间——买家不用担心买到义乌货 |
| 示范效应 | 其他市集开始模仿"原创审查"机制 |
| 批评 | 有人认为门槛太高——但乐天没有因为批评而降低标准 |

**100%原创规则是公共性的硬核表达**：公共性不是"谁都欢迎"，而是"我们选择站在哪一边"——乐天选择站在原创者一边，即使这意味着拒绝了一部分卖家。

### 3.3 国际连接：让景德镇重新进入全球对话

乐天陶社的国际艺术家驻场计划和国际交流，创造了一个**双向的公共通道**：

| 连接方向 | 内容 |
|----------|------|
| 国际 → 景德镇 | 全球陶艺家把当代观念、技术和方法带入景德镇 |
| 景德镇 → 国际 | 景德镇的材料、工艺和文化通过驻场艺术家传播到全球陶艺界 |
| 公共性 | 景德镇不再是"中国的陶瓷博物馆"，而是"全球陶瓷创新的现场" |
| 长期效应 | 国际艺术家回国后成为景德镇的传播者——不需要乐天花推广费 |

这不是"文化输出"的国家叙事，而是**创作者之间的横向连接**——一个美国陶艺家和一个景德镇师傅在同一个工作室里聊天，这才是真正的交流。

### 3.4 免费讲座与陶瓷教育：知识的公共化

| 维度 | 内容 |
|------|------|
| 持续年限 | 20年 |
| 费用 | 免费 |
| 内容范围 | 技术、观念、趋势——陶瓷相关的所有知识 |
| 公共性意义 | 在"知识付费"成为常态的时代，坚持"知识应该是免费的" |
| 不可替代性 | 一个年轻陶艺人如果没钱上培训班，至少还能来听免费的讲座 |
| 系统效应 | 降低了景德镇作为一个"学习场所"的进入门槛——吸引更多人成为"景漂" |

**免费讲座不是慈善，是公共基础设施**。就像公园和路灯一样，它的存在改变了整个城市的可及性。

### 3.5 公共性实践的三个层次

乐天陶社的实践在不同尺度上产生了公共效应：

```
层次 1：创作者与创作者之间
└── 周六市集让原创者们彼此看见、相互竞争、相互激励
    100%原创规则确保了"创作的质量驱动而非价格驱动"的场域

层次 2：创作者与城市之间
└── "景漂"让景德镇从一个手工业城市变成一个创意城市
    城市的空间、经济、人口结构都因这些人而改变

层次 3：城市与世界之间
└── 国际驻场计划让景德镇重新进入全球陶艺对话
    景德镇不再是"过去"，而是"现场"
```

### 3.6 边界与选择：什么不是公共性

乐天陶社的公共性有明确的边界，这些"不做的事"定义了它的硬度：

| 不做的事 | 原因 |
|----------|------|
| 不降低100%原创标准 | 即使有压力——原创是底线，不是口号 |
| 不把免费教育变成知识付费 | 即使可以盈利——教育是基础设施，不是产品 |
| 不依靠政府拨款 | 独立资金来源（创始人个人投资+运营收入）保持自主性 |
| 不把国际驻场变成"旅游项目" | 是严肃的创作交流，不是文化观光 |
| 不离开景德镇 | 即使其他城市提供更好的条件——在地性是核心 |

### 3.7 独有贡献：将"城市"确立为创意实践的画布

在 B Community 的案例中，乐天陶社贡献了一种独特的方法论：

> **一个组织可以通过持续在特定城市构建创意基础设施，改变整座城市的叙事和经济。**

这超越了"做公益"或"做企业社会责任"的层面。乐天陶社没有做"陶瓷慈善"，它做的是"让景德镇成为一个年轻人愿意来、可以创作、能够生存的城市"。这种公共性是结构性的，不是补充性的——它改变的不是一个社区里的几个人的命运，而是整个城市对于"自己可以是什么"的想象。
