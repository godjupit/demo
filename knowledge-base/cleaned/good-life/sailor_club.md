# 水手俱乐部 · 按良好生活标准分类

> **良好生活三问**：
> 1. 是否仍然愿意想象、表达、创造——在既有路径之外，把新的可能带进现实？
> 2. 是否在不确定的现实里，诚实地面对自己，找到内在自洽与韧性？
> 3. 是否在关照自我的同时，关切他人、回应周遭、参与塑造共同世界？

---

## 一、创造力 / Create Well

> *"仍然愿意想象、表达、创造，是在既有路径之外，在线性的成功学叙事之外，并把一些新的可能带进现实。"*

### 1.1 "玩"作为创作方法

水手俱乐部的创造力起点不是"做什么"，而是**"怎么玩"**。它的核心主张是将"玩"确立为一种严肃的创作方法——不是用创作取代玩，而是让玩本身就是创作。

> 食物、材料、游戏——日常经验就是创作入口。

这种方法的独特性在于：它不区分创作者和参与者。在 workshop 中，每个人都在"玩"，而正是在玩的过程中，新的表达和连接自然涌现。**"玩"消解了"专业创作者"与"普通参与者"之间的壁垒。**

| 创作入口 | 具体形式 |
|----------|----------|
| 食物 | 以食物为媒介的工作坊——味觉、触觉、分享都是创作 |
| 材料 | 手工材料——身体与物质世界的直接互动 |
| 游戏 | 游戏机制——规则内的自由、集体性的创造 |

### 1.2 跨媒介实践：工作坊 + 展览 + 播客

水手俱乐部不局限于单一媒介，而是将三种形式编织在一起：

| 媒介 | 功能 | 创作特征 |
|------|------|----------|
| **工作坊（Workshop）** | 核心创作场域 | 参与式、动手做、体验式——创作发生在"一起做"的过程中 |
| **展览（Exhibition）** | 成果展示与公共表达 | 将工作坊的成果转化为可观看、可进入的空间叙事 |
| **播客（Podcast）** | 连接人与日常经验 | 以声音媒介延展对话，将现场的"玩"转化为可传播的叙事 |

这种跨媒介实践与修四边形的方法论有相似的底层逻辑：**不是改良旧媒介，而是在媒介之间创造新的连接方式。**

### 1.3 "水手"之名：命名即姿态

> "水手"（Sailor）——旅程、探索、流动性。

命名本身就是一种创造性的自我定位：

| 名称的暗示 | 含义 |
|------------|------|
| **旅程** | 不是抵达某个终点，而是持续在路上的状态 |
| **探索** | 不预设结果，允许发现未知 |
| **流动性** | 不被固定身份、固定场所、固定形式束缚 |
| **俱乐部** | 不是机构、不是品牌——是一个大家"在一起"的空间 |

这与修四边形"修而非造"的命名逻辑形成了有趣的对照：修四边形强调的是**修补与撬动**，水手俱乐部强调的是**航行与发现**。两种姿态都是对"建造/生产"范式的替代。

### 1.4 与修四边形的方法论共鸣

水手俱乐部与修四边形在三个维度上共享价值观：

| 维度 | 修四边形（阿久） | 水手俱乐部 |
|------|-----------------|-----------|
| **游戏** | pervasive game——让游戏侵入真实世界 | "玩"作为创作方法——让创作变成游戏 |
| **社群** | 茶水间——为创作者建造的基础设施 | 工作坊——在共同制作中形成社群 |
| **公共空间** | 街头游戏——重新激活城市公共空间 | 城中村实践——在边缘空间创造连接 |

两者的共同信念：**通过"玩"，在日常生活的缝隙中打开新的可能性。**

## 二、身心安顿 / Live Well

> *"如何在不确定、不断变化、也并不总是友好的现实里，依然能够诚实地面对自己，慢慢找到一种内在自洽，同时保有韧性的生活方式。"*

### 2.1 在城中村安身：边缘作为位置

水手俱乐部扎根于广州康乐村（Kangle Village），一个典型的城中村（urban village）。这不是"找不到更好的地方"，而是**一种有意识的空间选择**。

> 在城市的缝隙中工作，本身就是一种立场。

| 空间选择的意义 | 说明 |
|----------------|------|
| **经济自洽** | 城中村的生活成本允许实验性的、不以盈利为首要目标的创作 |
| **边缘视角** | 不在CBD、不在创意园区——从边缘观察和参与城市 |
| **真实生活** | 城中村是日常生活密度最高的空间——食物、材料、人，都是创作资源 |

**吉一·青造社**作为水手俱乐部的实体据点，不是一个"工作室"或"展厅"，而是一个**让"玩"和"做"可以持续发生的地方。**

### 2.2 共同制作中的归属感

> "连接人和日常经验"——实践本身就是归属感的来源。

水手俱乐部的"身心安顿"不是通过独处或自我修炼实现的，而是**在共同制作的过程中自然生成的**：

- 一起做食物 → 分享味觉经验 → 建立了超越语言的连接
- 一起做手工 → 身体的共同在场 → 沉默中的彼此陪伴
- 一起玩游戏 → 规则中的自由 → 笑声中的彼此接纳

**这种归属感不依赖于宣言或身份标签，而是在具体的"做"中自然编织而成。**

### 2.3 "玩"作为自我修复

在一个以生产力和效率为衡量标准的社会里，"玩"常常被视为无用之事。水手俱乐部坚持"玩"的正当性，这本身就是一种**对功利主义时间伦理的抵抗**：

| 抵抗对象 | 水手俱乐部的回应 |
|----------|-----------------|
| 效率至上 | 不追求"产出"，只追求"玩得尽兴" |
| 专业壁垒 | 食物、手工、游戏——不需要专业训练就可以参与 |
| 社交表演 | 不需要"认识人"或"建立人脉"，一起做就够了 |

## 三、公共性 / Benefit Well

> *"如何在关照自我的同时，关切他人、回应周遭、关心公共，参与塑造我们共同身处的世界。"*

### 3.1 游戏 × 社群 × 公共空间：三维公共参与

水手俱乐部的公共性建立在三个维度的交织上：

```
        游戏
        /  \
       /    \
      /      \
     /  水手  \
    /  俱乐部   \
   /            \
  /              \
社群 ────────────── 公共空间
```

| 维度 | 公共性体现 |
|------|-----------|
| **游戏** | "玩"是普世的语言——不设门槛，任何人都可以参与 |
| **社群** | 工作坊不是单向教学，而是所有人一起创造——去中心化的共同在场 |
| **公共空间** | 城中村、街道、社区——在真实的城市肌理中，而非封闭的白盒子空间 |

### 3.2 在日常材料中连接彼此

水手俱乐部选择食物、材料、游戏作为媒介——这些是**所有人都拥有经验的东西**：

- **食物**：每个人都吃东西，都有关于食物的记忆和感受
- **材料**：手工材料是触觉的、直觉的，不需要翻译
- **游戏**：规则足够简单，所有人都可以立刻进入

这种"低门槛"不是妥协，而是**公共性的前提**：只有当参与不需要特殊知识或身份时，公共空间才真正向所有人敞开。

### 3.3 广州替代性创意生态的一环

水手俱乐部不是孤立的实践，而是广州替代性创意生态系统的一部分：

| 实践者 | 与水手俱乐部的关联 |
|--------|-------------------|
| **二高表演** | 同在广州，以身体和表演探索城市与身份 |
| **绣绣故事会** | 以叙事和手工连接社区 |
| **时代美术馆** | 广州当代艺术的重要节点 |

这个生态系统共享一种气质：**在小尺度上深耕，在日常中创造，在边缘处连接。**——不以规模或声量为目标，而是以关系的密度和质量为目标。

### 3.4 媒体可见性：界面新闻的报道

水手俱乐部被界面新闻（Jiemian News）报道，这意味着其实践获得了公共媒体的注目。这不是"出圈"的刻意操作，而是**实践本身的公共性自然吸引了外部的目光**——当你在城中村认真做食物、做手工、带人玩游戏，这件事本身就具有足够的叙事力量。
