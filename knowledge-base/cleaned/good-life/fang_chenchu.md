# 晨初 · 按良好生活标准分类

> **良好生活三问**：
> 1. 是否仍然愿意想象、表达、创造——在既有路径之外，把新的可能带进现实？
> 2. 是否在不确定的现实里，诚实地面对自己，找到内在自洽与韧性？
> 3. 是否在关照自我的同时，关切他人、回应周遭、参与塑造共同世界？

---

## 一、创造力 / Create Well

> *"仍然愿意想象、表达、创造，是在既有路径之外，在线性的成功学叙事之外，并把一些新的可能带进现实。"*

### 1.1 核心创造路径：从"业余者的长期主义"出发

晨初的创造力起点不是典型的"艺术家作品生产"，而是一种**对专业壁垒的系统性拆除**。"业余者的长期主义"——这个她用来描述自己实践方式的短语——包含了两个看似矛盾、实则统一的词：

> "业余者"拒绝权威身份，"长期主义"拒绝速成逻辑。

她的创造不是产出"自己的作品"，而是**创造一个让艺术可以被普通人触摸、参与、停留的场域**。这个场域就是假杂志的公共教育——一个独立摄影书出版机构，被她逐渐改造成了一个有温度、有触感的公共空间。

| 创造维度 | 具体内容 |
|----------|----------|
| 身份重构 | 不做"艺术家"，做"公教负责人"——让连接本身成为创造 |
| 媒介拓展 | 从摄影书出发，延伸到读书会、工作坊、城市行走、开春书会 |
| 场域创造 | 把一个出版品牌变成"可以触摸、参与、停留"的公共空间 |
| 叙事抵抗 | 不追求"个人作品"，而是让别人的创造在自己的场域中发生 |

### 1.2 发现新可能：让摄影书走出纸面

**转折点**：当假杂志从单纯的出版品牌转型为拥有实体图书馆和公共教育空间时，晨初找到了一种尚未被充分命名的创造形式——**公共教育作为艺术实践**。

一般艺术机构的公教部门通常是"配套"——展览的附属品。但晨初将公教本身当作**第一性的创造**：不是"先有展览，再做公教"，而是公教本身就是假杂志的核心产出。

**她创造的形态**：

| 形态 | 说明 | 创造性的体现 |
|------|------|-------------|
| 读书会 | 围绕摄影书的定期共读 | 不是单向讲解，是共创式阅读——每个人都在"完成"这本书 |
| 工作坊 | 动手制作、体验式的学习 | 让摄影和艺术从"观看对象"变成"身体经验" |
| 城市行走 | 在宁波的街巷中阅读城市 | 把整个城市变成一个可阅读的文本 |
| 开春书会 | 年度的摄影书市集+讲座+工作坊+城市行走 | 不只是书展，是一个社区围绕摄影书的年度狂欢 |
| 游击园艺 | 在城市的缝隙中种植物 | 将艺术实践延伸到最日常的城市干预当中 |

### 1.3 代表活动：开春书会——一年一度的"艺术公地"

**开春书会**是晨初创造力的集大成体现。它不是一个标准化的书展（几排桌子、卖书、走人），而是一个**多层次的社区事件**：

| 层次 | 内容 | 作用 |
|------|------|------|
| 摄影书市集 | 独立出版机构和个人展示、交换、售卖 | 经济层面：让独立出版可见、可变现 |
| 讲座/对谈 | 邀请摄影师、出版人分享 | 知识层面：打通"专业-业余"的信息差 |
| 工作坊 | 动手制作摄影书、暗房体验 | 身体层面：让参与者亲手做，而不只是看 |
| 城市行走 | 在宁波的街道中行走、观察、讨论 | 空间层面：重新感知自己生活的城市 |
| 游击园艺 | 在城市角落种植、照料植物 | 日常层面：艺术就是日常生活中的一件小事 |

**将所有这些封装在一个年度事件中——这就是晨初的"作品"**。她自己不站在聚光灯下，但她搭建的舞台让无数人的创造得以发生。

### 1.4 多条线索的编织者

晨初的创造力不只是在一个机构内，她同时在多重身份中穿梭：

| 身份 | 内容 |
|------|------|
| 假杂志公教负责人 | 规划和组织公共教育项目——读书会、工作坊、书会、行走 |
| boPOmofo 内容策展/编辑 | 独立出版和空间实践的内容策划 |
| 食物研究者 | 将对食物的研究融入文化和艺术活动中 |
| 植物爱好者 | "我很喜欢去观察，去触摸那些树、叶和花，触感会让我觉得，生命又重新在我身体里流动的感觉"——这种身体性的感知力渗透进她所有的公教设计中 |

这些线索不是彼此独立的"副业"，而是**在她身上自然编织在一起**——一场城市行走可能同时包含摄影、植物观察、食物体验。这就是"业余者的长期主义"的优势：不被专业边界分割，所以能将不同领域缝合起来。

### 1.5 创造的方法论：低姿态、高触感

从晨初的实践中，可以提取出她的创造方法论：

| # | 原则 | 说明 |
|---|------|------|
| 1 | **业余者即优势** | 不被专业话语绑架，所以可以说人话、做人事 |
| 2 | **触感先于概念** | "触摸树、叶和花"——身体感受是认知的基础，不是先有理论再去验证 |
| 3 | **场域先于内容** | 先搭好一个让人可以停留的空间，内容自然会长出来 |
| 4 | **连接先于表达** | 不是"我要表达什么"，而是"人和人之间因为什么能连起来" |
| 5 | **日常即道场** | 游击园艺、城市行走——在最日常的实践中埋下艺术的种子 |
| 6 | **低姿态的持续** | 不做大声量的"品牌活动"，而是让事情持续地、安静地发生 |

## 二、身心安顿 / Live Well

> *"如何在不确定、不断变化、也并不总是友好的现实里，依然能够诚实地面对自己，慢慢找到一种内在自洽，同时保有韧性的生活方式。"*

### 2.1 诚实面对：在"颠沛流离"中守住自己

假杂志经历过多次搬迁——"颠沛流离"不只是比喻，而是真实的机构生存状态。作为一个独立出版机构的成员，晨初面对的是一个持续不稳定的现实：租金压力、政策变动、公众认知有限。

在这种处境中，"业余者的长期主义"就显示出其内在力量——**正因为不把这件事当成"职业"，所以不会因为"职业受挫"而放弃**。业余者的身份成了一种保护：不管外部怎么变，她和艺术之间的关系不是由机构定义的，而是由她自己持续的投入定义的。

### 2.2 内在自洽：在创始人身后找到自己的位置

假杂志有一位辨识度很高的创始人言由——他是假杂志的公共面孔。晨初的角色是在幕后，做公共教育、运营社群、组织活动。这种"不站在聚光灯下"的位置，需要一种特定的内在自洽：

| 维度 | 内容 |
|------|------|
| 身份定位 | 接受自己不是"机构门面"，而是"连接者" |
| 满足感来源 | 不是来自公众认可，而是来自看到参与者在活动中获得共鸣 |
| 与创始人的关系 | 不是竞争，而是互补——言由建品牌，晨初建社区 |
| 创造力路径 | 不在美术馆做个展，而是在假杂志的空间中做让普通人可以参与的事 |

> "将摄影书和艺术从'墙上'拉到'生活中'的那个人"——这是晨初对自己的角色定位。这个定位本身就是自洽的：她不需要成为"墙上"的那个人，她的工作是把"墙上"的东西拉下来，让人们可以触摸。

### 2.3 韧性：植物般的生长节奏

晨初的自我描述中反复出现植物意象：

> "我很喜欢去观察，去触摸那些树、叶和花，触感会让我觉得，生命又重新在我身体里流动的感觉。"

这不是一个随意的比喻。植物的时间是不同于人类社会加速时间的另一种节奏——缓慢、沉默、但持续。晨初在假杂志的工作也是如此：不是爆发式的项目推进，而是一个读书会接一个读书会、一次城市行走接一次城市行走的日积月累。

**韧性不是"顽强抵抗"，而是植物式的"在条件允许的地方生长"**。假杂志从宁波出发，经历多次搬迁，但每次都重新扎根、重新开始公共活动。这种韧性不在于对抗，而在于适应后依然保持核心使命。

### 2.4 感官锚点：通过身体回到自己

晨初有一个非常具体的自我修复机制——**触摸植物**。当她感到生命力停滞时，她会去触摸树、叶和花，让触感唤醒身体中的流动感。

这个细节之所以重要，是因为它揭示了一种非语言的、身体性的自我关照。不是通过消费、娱乐或社交来"减压"，而是通过与自然的身体接触来**重置感知**。这种身体实践也与她的公教理念一致——她相信艺术需要被"触摸"、被身体体验，而不仅仅是被观看和思考。

### 2.5 价值观作为内在锚点

| 价值观 | 强度 | 如何在生活中体现 |
|--------|:---:|------|
| 业余者尊严 | ████ | "业余者的长期主义"——拒绝让专业壁垒隔开人与艺术 |
| 日常神圣 | ████ | 从触摸植物到游击园艺，在日常中找到意义 |
| 连接先于表达 | █████ | 让别人的创造发生，而非站在聚光灯下 |
| 慢即是快 | ████ | 不追求规模，追求持续——植物般的生长节奏 |
| 身体即认知 | ████ | 触感、行走、园艺——用身体而非概念来理解世界 |

## 三、公共性 / Benefit Well

> *"如何在关照自我的同时，关切他人、回应周遭、关心公共，参与塑造我们共同身处的世界。"*

### 3.1 公教即公共性：假杂志的"第三空间"

晨初的公共教育实践，本质上是**将假杂志从一个"出版品牌"变成一个"公共空间"**。这不是简单的业务扩展，而是一次机构性质的转变：

| 维度 | 原来的假杂志 | 晨初推动的转变 |
|------|-------------|---------------|
| 身份 | 独立出版品牌 | 出版品牌 + 图书馆 + 公共教育空间 |
| 受众 | 摄影爱好者和收藏者 | 任何对摄影和艺术感兴趣的普通人 |
| 与公众的关系 | 读者/消费者 | 参与者/共同创造者 |
| 空间的含义 | 办公室+仓库 | 一个可以被"触摸、参与、停留"的地方 |
| 知识流动 | 单向——出版→读者 | 多向——读书会中每个人都在参与 |

**假杂志的图书馆**是免费向公众开放的——这是一个重要的公共性姿态。在一个知识越来越被付费墙封闭的时代，一个可以自由翻阅摄影书的空间，本身就是一种公共性的声明。

### 3.2 降低门槛：每一种活动都是一个"入口"

晨初的公教项目有一个共同特征——**极低的参与门槛**：

| 活动 | 门槛 | 降低门槛的方式 |
|------|------|---------------|
| 读书会 | 不需要艺术史背景 | 以感受和讨论为主，每个人都可以发言 |
| 工作坊 | 不需要专业技巧 | 从动手体验出发，结果不重要 |
| 城市行走 | 不需要任何知识准备 | 穿着舒服的鞋来走就行 |
| 开春书会 | 免费或低票价 | 像一个社区节日，不是专业的博览会 |
| 游击园艺 | 不需要园艺经验 | 在城市的角落一起动手种 |

这种"低门槛"不是降低质量，而是**一种清晰的价值选择——不让专业壁垒成为人们接触艺术的障碍**。晨初在做的，正是她所说的：不让"专业"隔开人与艺术。

### 3.3 社区建设的三层结构

晨初通过假杂志的公教实践，在宁波建立了一个围绕摄影和艺术的创意社群：

```
第一层：核心社群
└── 经常参加读书会的本地摄影爱好者、艺术从业者
    在持续的共读和讨论中形成了深厚的智识连接

第二层：事件性社群
└── 开春书会吸引的全国独立出版人和参与者
    年度性地汇聚，形成一个临时的但高浓度的创作共同体

第三层：城市泛社群
└── 通过城市行走和游击园艺接触到的普通市民
    他们可能不知道"假杂志"是什么，但因为一场行走重新看到了自己的城市
```

这三层社群不是孤立的，而是**通过持续的公教活动彼此渗透**——参加城市行走的市民可能变成读书会的常客，读书会成员可能成为开春书会的志愿者。

### 3.4 宁波作为根基：一种有根的本土性

很多独立艺术空间选择去北京或上海，但假杂志坚持在宁波——一个并非当代艺术中心的城市。这个选择本身就是一种公共性的声明：

| 选择 | 公共性的意涵 |
|------|-------------|
| 留在宁波 | 不去"中心"，在"边缘"建设——让艺术的公共性在中心之外生长 |
| 开春书会在宁波 | 吸引全国做书人到宁波，而不是假杂志去全国各地参展 |
| 城市行走在宁波 | 让本地居民重新看见自己生活的城市——不是给游客做的 |
| 游击园艺在宁波 | 在本地城市的缝隙中种植——为这座城市的人创造微小的美好 |

**"假杂志的奇迹日常"**——这个概念捕捉到了在宁波这个看起来"不可能"的地方，让一个高品质的独立艺术空间持续运作所需要的日常努力。这个奇迹的基础不是资本或政策，而是晨初和团队日复一日的"做"：组织一场读书会、带一次行走、种一棵植物。公共性不是宣言出来的，而是做出来的。

### 3.5 B站「假象」系列：将公共性延伸到线上

假杂志在B站的「假象」系列访谈，采访了川内伦子（Rinko Kawauchi）、Alec Soth、冯立等摄影师，将假杂志的公共性从物理空间延伸到了数字空间：

| 维度 | 内容 |
|------|------|
| 形式 | 视频访谈/纪录片 |
| 受访者 | 国际和国内重要摄影师 |
| 平台 | B站——年轻人聚集的公共平台 |
| 门槛 | 比线下活动更低——免费、随时随地可看 |
| 意义 | 打破地域限制，让无法来宁波的人也能接触到假杂志的内容 |

虽然「假象」系列可能更多是言由主导，但作为公教负责人，晨初的工作与这种内容输出是一体的——都是在降低人们接触摄影和艺术的门槛。

### 3.6 她独特的公共性贡献：连接者而非输出者

在 B Community 的案例中，晨初贡献了一种容易被忽视但至关重要的公共角色：

> **连接者**——"将摄影书和艺术从'墙上'拉到'生活中'的那个人

不是创造新理论，不是产出自己的作品，不是建立自己的品牌。而是**把已经存在的好的东西——摄影书、艺术、知识——和需要它们的人连起来**。这种连接的公共性在于：它不为自己积累什么（名声、作品集、学术界地位），而是让流通发生。

在一个鼓励每个人都成为"IP"、都建立个人品牌的时代，选择做一个**连接者**——让聚光灯打在别人身上，让自己成为那个让聚光灯能亮起来的人——是一种反潮流的公共实践。

### 3.7 边界与选择：什么不是晨初的公共性

| 不做的事 | 原因 |
|----------|------|
| 不做"网红打卡点" | 假杂志的空间不是"出片场景"，而是可以停留和思考的地方 |
| 不追求规模 | 不做大声量的营销活动，公共性在于深度而非广度 |
| 不把自己包装成"意见领袖" | 出现在采访中是被访者，不是自我推广者 |
| 不让公教变成展览的附属 | 公教本身就是核心，不是"配套" |
