# 二高（何其沃） · 按良好生活标准分类

> **良好生活三问**：
> 1. 是否仍然愿意想象、表达、创造——在既有路径之外，把新的可能带进现实？
> 2. 是否在不确定的现实里，诚实地面对自己，找到内在自洽与韧性？
> 3. 是否在关照自我的同时，关切他人、回应周遭、参与塑造共同世界？

---

## 一、创造力 / Create Well

> *"仍然愿意想象、表达、创造，是在既有路径之外，在线性的成功学叙事之外，并把一些新的可能带进现实。"*

### 1.1 拒绝既有路径：从"中心"出走，创立"南方舞"

二高的创造力起点是**地理与美学层面的双重拒绝**。他从香港演艺学院毕业后，没有像大多数舞者那样进入北京或上海的当代舞蹈中心叙事，而是回到广州，在艺术体制的边缘地带开始创作：

> 拒绝的不是舞蹈本身，而是"谁有资格定义舞蹈"的权力结构。

他选择在广州的 grassroots 层面工作，主动远离中心化的资源分配体系和主流审美话语。这种拒绝不是逃避，而是**为另一种舞蹈叙事争取生存空间**。

| 拒绝对象 | 具体内容 |
|----------|----------|
| 北京/上海舞蹈中心叙事 | 不在体制内谋求地位，不进入主流舞蹈圈的晋升体系 |
| 西方现代舞正统 | 不以西方现代舞技法作为唯一标准 |
| 精英化的舞蹈话语 | 不追求"高雅""纯粹""专业"的审美标签 |
| 固定空间依赖 | 2023年关闭广州实体空间，转向游牧模式 |

### 1.2 发现新可能："土俗、山寨、南方舞"的美学策略

**核心创造**：二高提出"土俗、山寨、南方舞"三个关键词，并将其作为一套完整的创作方法论。

> "土俗、山寨、南方舞"是一套在中国南方生成的酷儿行动策略。

这不是对"低端"的猎奇式消费，而是在全球当代舞蹈的话语体系中，**以南方的本地经验为基点，创造一种反霸权的美学语言**：

| 关键词 | 内涵 | 创作体现 |
|--------|------|----------|
| **土俗** | 接地气的、乡土的身体语言，不追求"国际化"的优雅与抽象 | 从阳江民间文化、广东日常生活动作中提取身体素材 |
| **山寨** | 以低成本、模仿、挪用为策略，质疑原版/赝品的等级秩序 | 故意使用低技术手段、二手材料、"不专业"的呈现方式 |
| **南方舞** | 以珠三角为锚点的身体实践，区别于北方/西方中心主义 | 广州、顺德、香港——南方身体的迁徙路线与情感地理 |

**理论意义**："山寨"在这里被重新编码为一种批判性策略——不是无力做到"正版"的无奈，而是主动以"山寨"瓦解原创/复制的二元等级。这与阿久用 pervasive game 颠覆"电子游戏=游戏"的单一路径异曲同工。

### 1.3 代表作品：把新的身体叙事带进现实

| 作品 | 类型 | 年份 | 创造力的体现 |
|------|------|------|-------------|
| **《来来舞厅》** | 舞蹈剧场 | — | 以 Disco 文化和性别身份为入口，探讨改革开放后中国身体观念的变迁。舞厅作为边缘群体自我表达的公共空间 |
| **《恭喜发财》** | 舞蹈剧场 | — | 以乡愁和家庭为主题，将个人与父亲的情感张力转化为身体叙事 |
| **《蝴蝶岛》** | 沉浸式剧场 | — | 180分钟的沉浸体验，打破观众与表演者的固定关系，重新定义"看舞蹈"的方式 |
| **《躲猫猫》** | 共创项目 | — | 与视障社群深度协作一年后创作，视障者不是被"帮助"的对象，而是共同创造者 |
| **《步步高》** | 舞蹈剧场 | — | 以广州的非洲蜡染布料（African wax print fabric）为新媒介，探索全球化语境下南方身体的文化混杂 |

### 1.4 方法论的持续演化：从身体到媒介

二高的创造力不是静止的，而是在十八年间（2007-2026）持续演化的：

```
第一阶段（2007-2015）
└── 以身体为核心：舞蹈剧场作品，探索个人身份、家庭、性别

第二阶段（2015-2022）
└── 跨媒介扩展：合成现场概念形成，融入口述、影像、展览

第三阶段（2022-2023）
└── 田野转向：抖音田野考察——观看普通中国人如何在抖音上跳舞
    "K-pop 的动作是四处借鉴的——如果去掉标签，所有身体的律动都是相通的"

第四阶段（2023-至今）
└── 游牧与新材料：关闭实体空间→游牧集体
    布料（非洲蜡染）、刺绣（绣绣故事会）作为新的身体媒介
```

**抖音田野考察的意义**：2022年，二高进行了系统的"抖音田野考察"——不是批判或俯视，而是认真地观看普通中国人如何在短视频平台上使用自己的身体。他发现 K-pop 舞蹈动作其实是"四处借鉴"的，如果去掉文化标签，所有身体的律动本质上相互连接。**这是一种人类学式的创造力——在看似最表面的大众文化中发现身体政治的深层线索。**

### 1.5 命名即创造："二高"与"合成现场"

> "二高"这个名字，来自他在班上身高排第二——拥抱普通，拒绝光环。

**"二高"**的昵称本身就是一种姿态：不给自己取一个"艺术化"的名字，而是用一个中学班级里的身高排名作为身份标识。这种自嘲和普通感贯穿他所有的创作——不追求大师光环，而是保持一种"我就是普通人"的身体感。

**"合成现场"**（Synthetic Site）这个概念的提出，是对既有表演分类的超越：它不是一个舞剧、不是一场展览、不是一次行为艺术、不是一部纪录片——它是所有这些的"合成"。这个命名本身就是创造：**当一个现象无法被既有词汇描述时，你需要发明一个新词。**

## 二、身心安顿 / Live Well

> *"如何在不确定、不断变化、也并不总是友好的现实里，依然能够诚实地面对自己，慢慢找到一种内在自洽，同时保有韧性的生活方式。"*

### 2.1 诚实面对：与父亲的"误会"与和解

二高的故事核心有一条深沉的情感线索——他与父亲的关系：

> 父亲从未看过他的演出。这导致了"一定的误会"。

作为一个在广东阳江传统家庭中长大的男孩，选择以舞蹈为生、以酷儿身体为表达主体，注定要与父辈的期待产生张力。父亲的不看，既是一种缺席，也是一种沉默的"不同意"。

然而，这种"误会"并没有演变成决裂。父亲曾对他说：

> "我们都在做一些别人不理解的事情。"

这句话成为二高的力量来源。**父亲虽然没有看他的演出，但父亲理解"不被理解"——这是一种跨越代际的、以沉默传递的共情。**二高的"身心安顿"不是建立在冲突的解决上，而是建立在**学会与"不被理解"共存**的能力上。

### 2.2 内在自洽：游牧作为一种主动选择

2023年，二高表演关闭了广州的实体空间。这不是经营的失败，而是一种有意识的转变：

> "风吹鸡蛋壳，财散人安乐。"

他用一句广东俗语来描述这个决定——钱财散去，人反而轻松了。这表明关闭空间不是被动的"做不下去了"，而是主动选择**不被空间绑架**。

| 维度 | 关闭前（2007-2023） | 关闭后（2023-至今） |
|------|---------------------|---------------------|
| 空间形态 | 广州固定空间 | 游牧：广州、顺德、香港、伦敦 |
| 创作模式 | 场地依赖 | 灵活嵌入不同社区和语境 |
| 经济压力 | 固定租金+运营成本 | 项目制、轻量运营 |
| 心理状态 | "做了一个决定，安定下来" | "游牧的自由" |

**"到了大城市，记得要讲文明"**——这是二高表演2007-2023回顾展的自嘲式标题。它既是对自己的调侃（来自小城市的人到大城市要"装文明"），也是对城市化叙事的微妙批评（什么是"文明"？谁来定义？）。这种幽默感是内在自洽的重要标志：**能够调侃自己的处境，说明已经不再被它困扰。**

### 2.3 韧性：十八年的有机生长

从2007年在广州创立至今，二高表演已经持续运作了18年：

```
2007 ──┬── 香港演艺学院毕业（全奖），在广州创立二高表演
       │
2007-2015 ──┬── 从"接活儿来做"的小组织开始
            │    持续创作舞蹈剧场作品
            │
2015 ──┼── 合成现场概念逐步成形
       │
2020-2022 ──┼── 疫情期间的调整与反思
            │    抖音田野考察
            │
2023 ──┼── 关闭广州实体空间 ★ 重大转折
       │
2023-2026 ──┼── 游牧模式
            │    国际连接深化：挪威、意大利、瑞士、日本、香港、伦敦
            │    绣绣故事会启动
            │    南方舞厅持续运行
```

**韧性的特征**：
- **不追求线性增长**：不是"从小做大"，而是"在变化中持续"——2007年到2026年，从"接活儿来做"到国际认可，是缓慢的、有机的积累
- **适应变化的能力**：疫情、关闭空间、国际迁徙——每一个转折都没有让组织消失，而是催生了新的形态
- **身份的多重性**：舞者、编舞、社群组织者、田野考察者、游牧者——不把自己锁死在单一身份里

### 2.4 南方身体的迁移：生活在地理与情感之间

二高的生活轨迹——广州、顺德、香港、伦敦——不是旅游，而是一种**南方身体的迁移实践**。他反复穿行于珠三角的地图中，把这些城市之间的情感距离、文化差异、身体记忆编织进创作。

这种迁移本身就是"身心安顿"的方式：**安顿不在于"定下来"，而在于"知道自己在哪里、为什么在那里"。**

### 2.5 价值观作为内在锚点

从二高的实践中可以提取出以下核心价值观：

| 价值观 | 强度 | 如何在生活中体现 |
|--------|:---:|------|
| 边缘先于中心 | █████ | 拒绝北京/上海中心叙事，坚持在广州 grassroots 层面工作 |
| 普通先于精英 | █████ | "二高"昵称、抖音田野考察、南方舞厅的"谁都可以来" |
| 诚实先于美化 | ████ | 承认与父亲的"误会"、自嘲式展览标题、关闭空间时的坦然 |
| 持续先于规模 | █████ | 18年不间断创作，不求"做大做强"，求"持续在场" |

## 三、公共性 / Benefit Well

> *"如何在关照自我的同时，关切他人、回应周遭、关心公共，参与塑造我们共同身处的世界。"*

### 3.1 南方舞厅：身体版"茶水间"——为所有人打开的空间

**南方舞厅**是二高表演最具公共性的实践：一个社区舞蹈项目，口号是"谁都可以加入"。

| 维度 | 内容 |
|------|------|
| 形式 | 社区舞蹈——不分年龄、性别、职业，谁都可以来跳 |
| 精神 | 不是"我跳你看"，而是"我们一起跳"——打破表演者与观众的分隔 |
| 门槛 | 零门槛——不需要舞蹈基础，身体就是入场券 |
| 空间 | 不一定在排练厅——可以在公园、社区中心、榕树下 |
| 意义 | 把舞蹈从精英艺术还原为普通人的身体快乐 |

> "南方舞厅"不是表演项目，而是一个**持续开放的公共空间**。每一个人进入这个空间的人都同时是观看者和被观看者——舞蹈从专业化的"作品"回归到社会性的"连接"。

**与阿久的"茶水间"对照**：阿久为游戏创作者建立分享半成品的空间，二高为所有人建立分享身体的空间。两者的共同逻辑是——**降低门槛，让创造/表达成为普通人的权利，而非专业人士的特权。**

### 3.2 与视障社群共创：《躲猫猫》的深度协作

《躲猫猫》是二高表演公共性实践的典范：不是"为"视障社群做项目，而是"与"视障社群共创。

> 在与视障社群互动一年之后，才开始创作。

这个过程的关键在于**时间的投入**：不是一次性的"采风"或"调研"，而是长达一年的日常互动——在这个过程中，视障者不是被"帮助"的对象，不是被"代言"的弱势群体，而是**平等的合作者**。创作成为相互学习的载体：二高学习如何以非视觉的方式感知身体，视障者学习如何将自己的身体经验转化为表达。

| 传统公益/艺术模式 | 二高的协作模式 |
|-------------------|---------------|
| 艺术家"帮助"弱势群体 | 双方作为平等的合作者 |
| 短期项目制 | 一年以上的深度互动 |
| 艺术家"替"弱势群体发声 | 弱势群体直接参与创作过程 |
| 产出导向（作品） | 过程导向（关系建立） |

### 3.3 黄边EA店与绣绣故事会：嵌入社区的身体实践

二高表演的公共性不仅仅存在于"舞蹈圈"，而是深入到具体的社区中：

**黄边EA店**：在广州黄边村做的10天限时艺术空间——与村民一起在广场上跳舞。这不是"把艺术带到乡村"的姿态，而是**与村民一起用身体重新定义公共空间**。10天的时限本身就是一种"不占有"的承诺——不是来改造社区的，是来一起度过一段时间的。

**绣绣故事会**（EDPG 旗下的子项目）：将刺绣、讲故事和身体实践结合，在顺德的榕树下做免费工作坊。榕树是广东乡村传统的公共空间——**选择榕树而非美术馆，是公共性最明确的空间声明。**

| 项目 | 空间策略 | 参与门槛 | 公共性逻辑 |
|------|----------|----------|-----------|
| 南方舞厅 | 不限定的社区空间 | 零门槛 | 任何人都可以跳舞 |
| 黄边EA店 | 村中广场（10天限时） | 村民自然参与 | 与社区一起而非为社区 |
| 绣绣故事会 | 顺德榕树下 | 免费开放 | 传统公共空间+当代身体实践 |

### 3.4 "社区/社群作为工作方法"的方法论

二高表演提出了一个关键的方法论转向：

> 不是"building perspective"（建造视角），而是"dwelling perspective"（栖居视角）。

**建造视角**：从外部进入社区，带着蓝图和计划，改造、提升、输出——这是传统的社区艺术/社区营造的逻辑。

**栖居视角**：进入社区、住在其中、与社区一起生活——创作不是从"我想做什么"开始，而是从"我们在一起能做什么"开始。

这种视角转换是公共性的核心：**不是把社区变成作品的素材，而是让社区成为创作的主体。**

### 3.5 社交媒体作为公共基础设施

二高表演对社交媒体的使用不是追逐流量的营销逻辑，而是一种反算法的公共策略：

| 功能 | 如何使用 | 反主流逻辑 |
|------|----------|-----------|
| **文献库** | 将过往所有作品、田野笔记、访谈记录公开 | 不追求时效性，追求可查阅性 |
| **社群入口** | 任何人都可以通过社交平台找到二高表演并加入 | 不设置付费墙或会员制 |
| **活动召集工具** | 用社交媒体发布南方舞厅等活动的信息 | 不是为了涨粉，是为了让人来 |

这是一种**反流量逻辑**的社交媒体使用：不是努力让内容被更多人看到，而是让需要的人可以找得到。社交媒体不是终点（不是"成为网红"），而是工具（是"让人找到我们"）。

### 3.6 国际连接：南方身体的全球网络

二高表演的国际连接是有方向性的——不是"走向世界"的单向输出，而是**以南方为基点的双向交换**：

| 连接 | 内容 |
|------|------|
| 挪威 | 驻留与交流 |
| 意大利 | 合作与演出 |
| 瑞士 | 艺术家交换 |
| 日本 | 亚洲语境下的舞蹈对话 |
| 香港 | 回归南方网络的枢纽——因为香港演艺学院是二高的起点 |
| 伦敦 | 游牧期间的驻留 |

这些连接不是为"国际化"而国际化，而是**把南方身体实践放在全球语境中，让本地经验与世界对话**——不是为了被"世界认可"，而是为了构建替代性的知识交换网络。

### 3.7 边界与选择：什么不是公共性

二高的公共性同样有自己的边界：

| 不做的事 | 原因 |
|----------|------|
| 不做精英化的舞蹈作品 | 舞蹈是每一个人的身体权利，不是少数人的技艺 |
| 不做一次性的"公益"项目 | 与视障社群互动一年才开始创作——拒绝表演性的关怀 |
| 不做永久性的空间占有 | 黄边EA店只有10天——社区不是领土 |
| 不做以"帮助"为名的俯视 | 与视障者是合作不是帮助 |
| 不追求社交媒体的流量增长 | 社交媒体是工具，不是目的 |

### 3.8 独有贡献：将身体确立为公共性的媒介

在 B Community 的所有案例中，二高表演贡献了一种独特的公共性路径：

> **Body as commons（身体作为公共资源）**

不是通过话语、文字、政策来介入公共，而是通过**身体本身**。南方舞厅里，身体不是表达"我"的工具，而是连接"我们"的媒介。每个人带着自己的身体来，在共同的节奏中感受到彼此的存在——这种连接不需要解释，不需要翻译，它发生在语言之前。

**舞蹈在二高这里不是艺术门类，而是一种社会关系的生产方式。**
