# 徐艺函 · 按良好生活标准分类

> **良好生活三问**：
> 1. 是否仍然愿意想象、表达、创造——在既有路径之外，把新的可能带进现实？
> 2. 是否在不确定的现实里，诚实地面对自己，找到内在自洽与韧性？
> 3. 是否在关照自我的同时，关切他人、回应周遭、参与塑造共同世界？

---

## 一、创造力 / Create Well

> *"仍然愿意想象、表达、创造，是在既有路径之外，在线性的成功学叙事之外，并把一些新的可能带进现实。"*

### 1.1 创造力的起点：从个人痛苦到公共艺术实践

徐艺函的创造力起点是**极度私人的伤痛**。1989年生于连云港，母亲55岁因癌症去世，母亲生前最大的遗憾是"没有过爱情"——这个遗言成为徐艺函追问"爱是什么"的原点。她不是社会学家或女性主义学者式地从理论出发做研究，而是**从自己最痛的地方出发**，把私人的伤口转化为公共的艺术行动。

> "人类是在照护的时候，身体照护的时候学会怎么去爱的。" ——柯倩婷

这是徐艺函整个实践的理论基石：她不是在"发明"创造，而是在"发现"照护劳动中原本就存在但被遮蔽的创造性和爱的能力。

| 创造力起点 | 具体内容 |
|------------|----------|
| 母亲去世 | 母亲55岁因癌症去世，遗言"没有过爱情"——成为追问"爱"的原动力 |
| 个人情感危机 | 经历失败的感情关系、三次离婚——不是抽象地研究爱，而是在痛苦中摸索 |
| 癌症焦虑 | 母亲患癌的经历让她对自身健康产生焦虑——照护者的恐惧成为创作材料 |
| 艺术媒介转型 | 从画廊里的陶瓷艺术家转向社区公共艺术实践——拒绝"纯艺术"的安全区 |

### 1.2 照护劳动公共化：一条尚未被命名的艺术路径

徐艺函的核心创造力在于**把"照护劳动"从私人家庭领域搬到公共空间**，使其成为可见的、可讨论的、可集体参与的艺术形式。这是一种媒介层面的创造：不是改良陶瓷艺术，而是把整个"伺候人的活"确立为艺术媒介。

| 实践路径 | 说明 |
|----------|------|
| 抹布神艺术节 | 收集家庭抹布，缝制成巨型蝴蝶，在村庄游行 |
| 保洁员靠垫项目 | 与上海外滩保洁员一对一合作，将她们的童年故事缝制成靠垫 |
| "母亲的选择"写作工作坊 | 为母亲/女性照护者创造书写自身经验的空间 |
| "不要分离"展览 | 从参与者家中收集物件，将她们的言语绣在窗帘上 |

### 1.3 代表作品：把新的可能带进现实

| 作品 | 类型 | 年份 | 创造力的体现 |
|------|------|------|-------------|
| **抹布神艺术节** | 社区公共艺术 | — | 从家家户户收集来的抹布——最卑微的家务工具——缝成巨型蝴蝶。在村庄里游行，开设"照护市集"。把"伺候人的活"从后台推到前台 |
| **保洁员靠垫项目** | 一对一合作艺术 | — | 与上海外滩保洁员一对一合作，每个靠垫承载一个童年故事。保洁员不是"被帮助的对象"，而是共同创作者 |
| **"母亲的选择"写作工作坊** | 社区写作 | — | 为妈妈们和女性照护者创造写作空间——不是教她们写作，而是让她们的经验被听见 |
| **"不要分离"展览** | 参与式装置 | — | 从参与者家中收集物品，将她们的言语绣在窗帘上——"家"的碎片被带入公共空间 |

### 1.4 创作方法论：五步协作流程

从徐艺函的实践中，可以提取出系统的创作方法：

| # | 步骤 | 说明 |
|---|------|------|
| 1 | **分享照护经验** | 先坐下来聊天——保洁员讲她们的故事，参与者分享自己的照护经历。不预设主题，让真实的经验浮现 |
| 2 | **收集抹布/物件** | 从家庭中收集使用过的抹布、日常物件——这些物品承载着真实的劳动痕迹 |
| 3 | **田野观察** | 进入保洁员的工作现场，观察她们如何与城市空间互动 |
| 4 | **共同创作** | 一对一合作——不是艺术家为她们创作，而是和她们一起创作。靠垫上的图案来自保洁员自己的童年记忆 |
| 5 | **重建连接** | 将作品带回公共空间展示，让不同阶层的人围坐在一起——重建被城市空间割裂的人与人之间的连接 |

### 1.5 命名的创造力："抹布神"与"起风了"

**"抹布神"**：将最卑微的家务工具——抹布——封"神"，这是一种创造性的价值重估。在中国民间宗教中，每一行都有守护神，而"抹布神"是徐艺函为照护劳动者发明的神——用荒诞的庄重，为看不见的劳动赋予神圣性。

**"起风了"**：社区名称来自"风起于青萍之末"——微小的行动可以汇聚成风。这不是"改变世界"的宏大叙事，而是承认自己的实践从最细微处开始。

> 陶瓷艺术从画廊走向社区，从个人表达走向共同创作——不是放弃艺术，而是扩展艺术的边界。

## 二、身心安顿 / Live Well

> *"如何在不确定、不断变化、也并不总是友好的现实里，依然能够诚实地面对自己，慢慢找到一种内在自洽，同时保有韧性的生活方式。"*

### 2.1 诚实面对：从个人危机中出发

徐艺函没有把自己包装成"公益人"或"女性主义者"的英雄形象。她的故事起点是一系列具体的、私人的痛苦：

| 危机 | 诚实程度 |
|------|----------|
| 母亲55岁癌症去世 | 不是抽象地"关心女性议题"，而是母亲具体的死亡和遗言击中了她 |
| 癌症焦虑 | 因为母亲患癌，她对自身健康产生恐惧——不回避自己的身体焦虑 |
| 三次离婚 | 不回避"爱情很难"这一事实——她自己也没有找到完美的答案 |
| 失败的感情关系 | 承认自己也在"爱"这件事上不断受伤和摸索 |
| 艺术身份危机 | 从陶瓷艺术家转向社区实践——承认传统艺术路径无法回应自己的生命问题 |

> "如果说我没有照顾我小孩的话，不知道我能不能够共情到一个躺在床上的人。"

这句话揭示了她的诚实：她不是站在道德高地上"帮助他人"，而是承认自己的照护经验（照顾孩子）是她能理解他人痛苦的前提。**照护不是美德表演，而是共情能力的来源。**

### 2.2 内在自洽：劳动是一种爱情

徐艺函的内在自洽来自一个深刻的认识：**照护劳动本身是学会爱的途径**。这个认识来自柯倩婷的理论，但在她的实践中被验证：

- 她的孩子说"劳动是一种爱情"——这句话成为她实践的注脚
- 通过照顾他人（孩子、母亲、社区参与者），她在疗愈自己失去母亲的伤
- 她不是在"付出"，而是在"学习"——学习怎么去爱

**这种自洽不是独善其身的，而是关系性的**：她的安顿不来自独处或冥想，而来自与他人的照护关系中。

### 2.3 韧性：六年的信任建设

徐艺函对待时间的耐心是她韧性的核心证据。以薛阿姨为例：

```
认识薛阿姨 ──→ 6年
开始工作坊 ──→ 3年后她才开始写作

不是"做了工作坊就有成果"的逻辑
而是"等到她愿意写"的耐心
```

**时间伦理特征**：
- **以年为单位**：不是"三个月出成果"的项目逻辑，而是以年为刻度积累信任
- **不追求可见产出**：三年工作坊薛阿姨才开始写作——在主流评价体系中，这是"低效"的
- **信任先于表达**：先让参与者感到安全，再让表达自然发生

### 2.4 从照护中疗愈：重建"公共性的家庭"

徐艺函的疗愈路径是独特的：

| 转折 | 内容 |
|------|------|
| 初始困境 | 母亲去世、癌症焦虑、三次离婚、"城市中的人如何去爱"的困惑 |
| 发现路径 | 通过与薛阿姨的关系，看到自家婚姻中的同构的阶级结构 |
| 核心领悟 | "城市中的人如何去爱"——问题不在于缺乏爱，而在于缺乏"公共性的爱" |
| 实践方案 | 建设"起风了社区"——一个超越血缘关系的"公共性的家庭" |
| 疗愈结果 | 不是找到答案，而是找到和他人一起追问的方式 |

> "城市中的人如何去爱"——她的回答是：**不是在私密关系中寻找完美的爱，而是通过照护劳动和公共实践，扩大"爱"的边界。**

### 2.5 价值观作为内在锚点

从徐艺函的实践中，可以提取出她的核心价值观：

| 价值观 | 强度 | 如何在生活中体现 |
|--------|:---:|------|
| 真实先于表演 | █████ | 坦诚三次离婚、坦诚自己对爱的困惑——不表演"完美公益人" |
| 关系先于产出 | █████ | 六年认识、三年工作坊才等到薛阿姨动笔——关系比成果重要 |
| 照护先于效率 | █████ | 不以项目产出为尺度，以信任建立为尺度 |
| 平等先于助人 | █████ | 保洁员是"共同创作者"而非"帮扶对象" |
| 公共先于私人 | ████ | 将私人情感经验转化为公共艺术行动 |

## 三、公共性 / Benefit Well

> *"如何在关照自我的同时，关切他人、回应周遭、关心公共，参与塑造我们共同身处的世界。"*

### 3.1 让看不见的劳动被看见：保洁员成为共同创作者

徐艺函公共性实践的核心是**让照护劳动从"不可见"变为"可见"**。她选择了城市中最具象征性的群体——保洁员：

| 维度 | 内容 |
|------|------|
| 问题意识 | 保洁员触摸城市的每一个角落，却不属于城市——她们是城市空间中最"不可见"的人 |
| 实践方式 | 一对一合作——艺术家和保洁员坐在一起，平等地创作 |
| 创作内容 | 靠垫——一个承载"家"和"照护"意象的物品。每个靠垫的图案来自保洁员自己的童年故事 |
| 身份转换 | 保洁员从"被服务的对象"变成"共同创作者"——不是帮助她们，而是和她们一起工作 |
| 理论意指 | "照护劳动公共化"——把家庭内部的、无偿的、不被认可的劳动，带入公共视野，赋予其价值和尊严 |

### 3.2 跨阶层连接的"童话时刻"

徐艺函的实践创造了在正常社会秩序中不可能发生的相遇：

> "在城市空间当中，不太可能肩并肩坐在一起干活的人，却围坐在了一起。"

一位参与者将这个场景描述为"像童话一样"——银行职员、研究生、保洁员坐在一起缝制靠垫。这不是刻意设计的"团建"，而是通过共同劳动自然发生的连接。

**"童话"的含义**：在阶层分化的城市日常中，这些群体几乎没有平等相处的物理空间。徐艺函的艺术实践创造了这样一个空间——不是通过"对话"或"讲座"，而是通过**一起用手劳动**。

### 3.3 薛阿姨的故事：照护者权利的隐藏代价

徐艺函通过长期的陪伴关系，揭示了照护劳动背后系统性的权利让渡。薛阿姨——一位家政工作者——的故事揭示了照护者在家庭中经历的**三重权利让渡**：

```
第一层：劳动权的让渡
└── 家政劳动被视为"帮忙"而非"工作"，报酬被压低，权益不被承认

第二层：情感权的让渡
└── 照护者被期待"像家人一样"付出情感，但情感付出不计入报酬

第三层：话语权的让渡
└── 照护者的经验和知识不被正视——"伺候人的活"被认为不需要智慧
```

**徐艺函的公共性贡献**：不是替薛阿姨说话，而是创造了一个让薛阿姨自己能够开口的空间。六年的相识、三年的工作坊，终于等到薛阿姨开始写作她自己的故事。

### 3.4 上海外滩项目：在阻力中建立连接

上海外滩靠垫项目的实施过程本身就是公共性实践：

| 挑战 | 应对 |
|------|------|
| 管理方的阻挠 | 管理方不希望保洁员与外来者接触——正常的权力结构试图隔绝这种连接 |
| 找到"关键连接者" | 发现保安队长才是能连接所有保洁员的人——社会网络有自己的关键节点 |
| 与保洁员交朋友 | 不是"进入现场→收集材料→离开"的采风逻辑，而是建立真实的友谊 |
| 共同创作而非提取 | 保洁员的童年故事缝在靠垫上——不是艺术家"再现"她们的故事，而是一起创作 |

**公共性不是没有阻力**：管理方的阻挠恰恰说明这种连接触及了真实的权力结构。真正的公共实践不是绕开阻力，而是在阻力中坚持。

### 3.5 "公共性的爱"：重新定义爱的范畴

徐艺函从个人追问"爱是什么"出发，最终抵达了一个公共性的答案：

| 阶段 | 对"爱"的理解 |
|------|-------------|
| 起点 | 母亲遗言"没有过爱情"——爱是浪漫关系的缺失 |
| 追问 | "城市中的人如何去爱"——发现私密的爱情远远不够 |
| 发现 | 通过照护劳动学会共情——"劳动是一种爱情"（她孩子的话） |
| 升华 | "公共性的爱"——不只是浪漫关系，而是扩展到社会的照护网络 |
| 实践 | 起风了社区——一个超越血缘关系的"公共性的家庭" |

> "如果说我没有照顾我小孩的话，不知道我能不能够共情到一个躺在床上的人。"

**照护经验是共情能力的基础**——这是徐艺函经过自身验证的洞察。公共性的爱不是抽象的口号，而是在具体的照护行为中建立的、可以扩展到陌生人的共情能力。

### 3.6 独有贡献：将"照护劳动"确立为公共艺术的媒介

在 B Community 的所有案例中，徐艺函/起风了社区贡献了一种极其独特的方法论：

> **Care Work as Public Art（照护劳动即公共艺术）**

不是用艺术"表现"照护，不是用艺术"帮助"照护者，而是**让照护劳动本身成为公共空间中的艺术行动**。抹布的缝制、靠垫的共创、写作工作坊——这些都是照护劳动的形式，同时也是艺术创作的形式。

**柯倩婷的总结**：
> "人类是在照护的时候，身体照护的时候学会怎么去爱的。"

**这不是关于"做艺术"，而是关于通过照护劳动重建人与人之间的爱。** 抹布不是隐喻，照护不是题材——抹布就是照护，照护就是艺术。这个等式本身，就是徐艺函带给公共艺术领域最激进的贡献。
