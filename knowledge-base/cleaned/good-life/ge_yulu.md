# 葛宇路 · 按良好生活标准分类

> **良好生活三问**：
> 1. 是否仍然愿意想象、表达、创造——在既有路径之外，把新的可能带进现实？
> 2. 是否在不确定的现实里，诚实地面对自己，找到内在自洽与韧性？
> 3. 是否在关照自我的同时，关切他人、回应周遭、参与塑造共同世界？

---

## 一、创造力 / Create Well

> *"仍然愿意想象、表达、创造，是在既有路径之外，在线性的成功学叙事之外，并把一些新的可能带进现实。"*

### 1.1 直接行动：不等许可的创造方法论

葛宇路的创造力起点是**直接行动（Direct Action）**——一种来自 David Graeber 的思想资源，核心信条是：不等别人给你许可，自己先把问题解决了。

> "直接行动的意思就是——你不要等别人来给你解决问题，不要等政府、等机构、等权威来批准你，你自己去做。"

这不是鲁莽，而是一种经过理论反思的创造姿态。在当代中国的语境下，这种"不等许可"的姿态本身就是一种创造性突破——它用行动而非宣言，重新定义了个人与公共空间、个人与制度之间的关系。

| 直接行动的核心特征 | 具体内容 |
|----------|----------|
| 不等待 | 不等政府命名道路，自己去命名；不等机构批准展览，自己创造展览条件 |
| 不抱怨 | 不批评系统、不控诉制度，而是用行动去"加"一些东西 |
| 不依赖 | 不依赖画廊体制、不依赖美术馆邀约、不依赖艺术基金 |
| 不解释 | 行动本身即是表达——不需要"艺术家陈述"来赋予合法性 |

### 1.2 发现新可能：艺术作为"连接"的工具

**转折点**：从中央美术学院毕业后，葛宇路没有走"画廊—藏家—美术馆"的常规艺术路径，而是选择了街道、城市监控系统、公共空间作为他的媒介。

> "我是想用艺术去加一切，去达成一个很多元的，每个人都有自己的空间的世界。"

他把艺术从白盒子里解放出来，放进真实的社会肌理——道路、摄像头、电网、市集。这是一种**媒介层面的创造**：不是改良旧媒介（架上绘画、雕塑、影像），而是发明新的艺术发生方式——**让生活本身成为艺术的材料和现场**。

**理论根基**：葛宇路的实践与情境主义国际（Situationist International）的"异轨"（detournement）和"漂移"（derive）传统有深层呼应，同时也与中国本土的"街头智慧"结合：

| 子类型 | 说明 | 葛宇路案例 |
|--------|------|-------------|
| 空间命名权 | 通过命名重新定义空间归属 | "葛宇路"路牌 |
| 制度介入 | 将冰冷的制度转化为有温度的人际接触 | 对视监控摄像头 |
| 身体能量转换 | 将身体劳动转化为可见的能源和数据 | 骑车发电供展览 |
| 身份交换 | 通过交换社会角色揭示隐藏的结构 | 与美术馆工作人员换身份 |
| 商业掩护 | 用商业活动的形式做公共服务 | 市集、节庆、定期聚会 |

### 1.3 代表作品：把新可能带进现实

| 作品 | 类型 | 年份 | 创造力的体现 |
|------|------|------|-------------|
| **"葛宇路"路牌** | 空间命名 | 2013 | 在百子湾南一路悬挂写着自己名字的路牌，被高德地图正式收录，引发交管、外卖、快递等城市系统的自我更新。一个人命名了一条路——用最低成本撬动最大系统效应 |
| **对视监控** | 制度介入 | — | 搬梯子爬到监控摄像头前，盯着镜头看，直到保安出来。用目光穿透监控系统，把"被观看"的关系反转——摄像头后面是一个可以坐下来聊天的人 |
| **骑车发电** | 身体能量 | — | 从河北燕郊骑行30公里到展览现场，把自己的体力转化为电能，只为维持展览屏幕运行30分钟。诚实地呈现个体能量的有限性——但有限不等于无力 |
| **与美术馆换身份** | 身份交换 | — | 在美术馆工作了3个月，让原本的工作人员获得9周带薪休假。证明了一个最朴素的道理：艺术需要闲暇——如果你没有时间喘息，你就没有时间创造 |
| **市集与节庆** | 商业掩护 | 2022至今 | 搬到成都后，从纯艺术转向市集、节庆、定期聚会——"用商业逻辑掩护公共表达"，在合规框架内创造自由空间 |

### 1.4 设计方法论：直接行动的六条原则

从葛宇路的实践中，可以提取出一套创造方法论：

| # | 原则 | 说明 |
|---|------|------|
| 1 | **行动先于声明** | 先挂路牌，再让世界发现它。不是先写方案申请，再执行——方案就是行动本身 |
| 2 | **最低成本，最大涟漪** | 一块路牌引发了城市的系统性回应——投入/产出比的极限 |
| 3 | **以连接代替对抗** | 对视监控不是破坏监控，而是"脱掉制服后坐在一起聊天"——瓦解冷漠而非攻击系统 |
| 4 | **诚实地呈现局限** | 骑车30公里只能发电30分钟——不用虚假的震撼来包装，把局限本身变成表达 |
| 5 | **利用系统的缝隙** | 不挑战系统，而是在系统允许的模糊地带行动 |
| 6 | **商业作为掩护，公共作为目的** | 用"我们只是想赚钱"来解释市集——其实是在创造公共空间 |

### 1.5 命名即创造：从"葛宇路"到"元元的菜地"

> "谁在做或是支持突破观念、松动规则、弱化唯一价值观，在固化的社会搭建交流桥梁，做激活个人创造性和能动性的事情，谁就是我的好朋友。"

**葛宇路**的名字本身就是一种创造性姿态——不是等待权力机构给你一个路名，而是自己去命名，让日常生活系统（外卖、快递、高德地图）来"批准"这个名字。当路牌最终被拆除，这成为作品的一部分——接受一切结果，包括失败和消失。

更重要的是，他后来在成都做的市集被当地居民自发称为"元元的菜地"——**命名权从艺术家转移到了普通人手中**。这是"葛宇路"作品最深刻的延续：不是让大家都挂自己的名字，而是激活每个人命名和创造自己世界的意识。

## 二、身心安顿 / Live Well

> *"如何在不确定、不断变化、也并不总是友好的现实里，依然能够诚实地面对自己，慢慢找到一种内在自洽，同时保有韧性的生活方式。"*

### 2.1 诚实面对：承认个体的"极其有限"

葛宇路的故事起点同样不是英雄式的叙事，而是一种**坦然承认局限的姿态**：

> "不好意思，我的状态就是这个样子，能量就这么多。"

> "一个个体在一个庞大的城市里面是极其有限的，我希望大家看见这种局限性。"

骑车30公里只够支撑展览30分钟——这不仅是作品的一部分，更是葛宇路对自己的诚实定位。他不扮演"改变世界"的英雄艺术家，而是诚实地展示：**一个人的身体能量、社会能量、政治能量都是有上限的。** 这种诚实恰恰是他的力量来源——因为不装强大，所以不必维护虚假形象，反而获得了真正的行动自由。

**这种诚实是"身心安顿"的前提**：如果不承认自己的能量有限，就会被耗尽；如果不承认路牌会被拆除，就会被挫败压垮；如果不承认体制的强大，就会陷入无用的对抗或被收编。

### 2.2 内在自洽：搬迁与重新定位

从北京到成都的搬迁不是"逃离"，而是一次**主动的重新定位**：

| 维度 | 北京时期 | 成都时期 |
|------|------|------|
| 空间 | 高度政治化、高成本、高压力 | 更多实验性空间、更低成本 |
| 方式 | 纯艺术行动（路牌、监控、发电） | 商业掩护 + 公共表达（市集、节庆、聚会） |
| 身份 | 独立"搞事情"的艺术家 | 社群组织者、市集运营者 |
| 风险规避 | 直接面对 | "这就是个商业活动，我们只是想赚钱" |

这种转变不是妥协，而是一种**策略性的内在自洽**——在规则日益收紧的环境中，用商业的外壳保护公共的内核。正如他所说：

> "这就是个商业活动，我们只是想赚钱。"

这句话本身就是一种**保护色**——在"疑罪从有"的环境里，最好的辩护不是证明自己无罪（那太难了），而是让指控本身失去靶心。

### 2.3 韧性：拥抱不确定性与不完美

葛宇路的韧性的一个关键面向是：**把失败和消失变成作品的一部分**。

- "葛宇路"路牌最终被城管拆除——这并未"终结"作品，而是完成了作品。从命名到被清拆，整个生命周期就是一件完整的作品
- 骑车发电的"不够用"——恰恰是作品的核心理念：诚实呈现个人的局限性
- 对视监控没有"改变"监控系统——但它改变了一个保安和一个艺术家之间的关系

这种接受不完美的能力，是葛宇路内在韧性的核心。**不是"战胜"不确定性，而是和不确定性一起生活。**

```
北京时期                      成都时期
    │                              │
    ├── "葛宇路"路牌 ★              ├── 市集运营
    │                              │
    ├── 对视监控                    ├── 节庆活动
    │                              │
    ├── 骑车发电                    ├── 定期聚会
    │                              │
    ├── 与美术馆换身份              ├── "元元的菜地"
    │                              │
    └── 路牌被拆 ★ 作品完成          └── 用商业逻辑掩护公共表达
```

**时间伦理特征**：
- **非积累式**：每一个作品完成自己，不追求"越做越大"
- **拥抱消失**：作品的生命周期包括其被拆除的过程
- **流动性**：从北京到成都，从纯艺术到市集商业——地理和形式都流动，但核心不变

### 2.4 自我保护：在"疑罪从有"时代的生存策略

葛宇路的生活智慧体现在他对自己所处环境的清醒认知：

| 策略 | 具体做法 |
|------|------|
| **商业包装** | 用"这就是个商业活动"来为公共活动获取合法性 |
| **不解释** | 不发表政治声明、不写宣言——减少被解读为"别有用心"的风险 |
| **地理迁移** | 从北京（"疑罪从有"更严格）到成都（更宽松的实验空间） |
| **社群化** | 不再单独行动，而是建立社群——人多不是"个人行为"，而是"市场行为" |

这些策略共同构成了一套在紧缩空间中保持创造力和公共性的**生存技术**。

### 2.5 价值观作为内在锚点

在葛宇路的实践中，可以提取出以下核心价值观：

| 价值观 | 强度 | 如何在生活中体现 |
|--------|:---:|------|
| 行动先于话语 | █████ | 挂路牌、搬梯子、骑车发电——用行动而非宣言表达 |
| 诚实先于震撼 | █████ | 承认个体有限、承认作品不完美、不包装 |
| 连接先于批判 | █████ | 对视监控是为了认识背后的人，不是为了控诉监控 |
| 游戏先于对抗 | ████ | 在系统的缝隙中玩，而不是正面冲击系统 |
| 激发先于主导 | █████ | 让每个人自己命名、自己创造——不是教别人怎么做 |

**这些价值观不是写在纸上的标语，而是每一次行动的自然流露**——当葛宇路挂上路牌、爬上梯子、骑行30公里、在市集上观察普通人的反应时，他用自己的身体兑现了这些价值。

## 三、公共性 / Benefit Well

> *"如何在关照自我的同时，关切他人、回应周遭、关心公共，参与塑造我们共同身处的世界。"*

### 3.1 路牌：一个人的行动如何变成公共基础设施

"葛宇路"路牌是葛宇路的公共性思想的起点和最纯粹的表达：

| 维度 | 内容 |
|------|------|
| 形式 | 在无名道路悬挂自己名字的路牌 |
| 系统回应 | 被高德地图正式收录、纳入导航系统 |
| 公共效果 | 快递员在这里送快递、交警在这里开罚单、外卖员送到这条路——"葛宇路"成为数以万计普通人的日常经验的一部分 |
| 拆除 | 最终被城管拆除——但拆除本身也进入了公共话语 |
| 意义 | 一个人的私人行动被城市系统"认可"为公共事实 |

> 一个人的名字，变成了一条可以被导航的路。

**路牌的公共性意义**：它证明了**一个人的行动可以穿透整个公共系统**——不是靠权力，不是靠金钱，不是靠关系，而是靠对系统运作逻辑的理解和被动的等待。葛宇路只是挂了块牌子，是高德地图的算法、快递员的地址簿、交警的罚单本共同完成了"葛宇路"的公共存在。

### 3.2 对视监控：把冷系统变回热的人

对视监控摄像头这个作品，是葛宇路公共性思想最具温度的体现：

**操作**：搬梯子爬到监控摄像头前，盯着镜头看，直到保安出来。

**结果**：
> "他脱下了制服，我们成为坐在一起聊天的朋友。"

**层次分析**：
- 第一层：用一个具体的"人"的目光，去面对一个抽象的"系统"的眼睛
- 第二层：当保安出来时，他们有了面对面的交流——"你为什么看监控？""因为这是我的工作。""你的工作无聊吗？""挺无聊的。"
- 第三层：两个人坐在一起，成为朋友——**系统没有被推翻，但人与人之间的墙裂开了一道缝**

这不是"对抗"监控系统，而是**穿透**监控系统——不是要拆除摄像头，而是要看到摄像头后面坐着一个无聊的人，这个人脱下制服后可以和你坐在一起聊天。

### 3.3 关系重建的三个层次

葛宇路的实践在不同尺度上重建了人与人的连接：

```
层次 1：人与城市系统之间
└── "葛宇路"路牌：个人行动 → 城市基础设施的回应
    导航系统、快递员、交警共同完成了这个公共事实

层次 2：人与"系统代理人"之间
└── 对视监控：冷系统 → 热的人
    保安脱下制服 → 两个人坐在一起聊天 → 朋友

层次 3：人与人之间（多元呈现）
└── 市集与节庆：邀请工厂诗人、社区阿姨
    "看见另外一个人是怎么在这个世界生活的"
```

### 3.4 让"看不见的人"被看见："重新回到公共"

搬到成都后的市集和节庆，标志着葛宇路公共性思想的深化——从艺术家个人的直接行动，转向**搭建平台让更多人被看见**：

| 实践 | 内容 | 公共性效应 |
|------|------|------|
| **邀请工厂诗人** | 让工厂工人出身的人在节日活动上朗诵诗歌 | "让大家看见另外一个人是怎么在这个世界生活的" |
| **社区阿姨参与（通过徐艺函）** | 让从未离开过村庄的社区阿姨在市集上展示和售卖 | 给予"没有被看见过的人"一个平台 |
| **市集运营** | 用商业形式创造公共聚集空间 | 让不同阶层、不同背景的人在同一空间相遇 |
| **定期聚会** | 建立持续的社群连接 | 不是一次性事件，而是持续的关系网 |

> "重新回到公共，让大家看见更多丰富有趣的、多样的人。"

**这句话是葛宇路公共性思想的核心**。它不追求某种宏大的社会变革，而是追求在日渐碎片化、原子化的社会中**重新创造人与人之间的看见和连接**。工厂诗人被邀请来朗诵，社区阿姨被请来摆摊——这些不是"艺术家提携弱势群体"的慈善叙事，而是**让不同的人生经验有一个互相照面的机会**。

### 3.5 国际连接：独立艺术家的跨文化实践

葛宇路的实践同样具有国际维度：

| 连接 | 内容 |
|------|------|
| 学术背景 | 中央美术学院，受过完整的学院教育 |
| 国际艺术界关注 | "葛宇路"路牌在国际艺术媒体和社交平台引发广泛讨论 |
| 艺术市场关系 | 作品进入国际画廊和收藏体系 |
| 理论资源 | David Graeber 的"bullshit jobs"和"直接行动"；情境主义国际的影响 |

### 3.6 边界与选择：什么不是公共性

葛宇路的公共性同样有明确的边界：

| 不做的事 | 原因 |
|----------|------|
| 不做政治宣言 | 不发表批判性文字，不用"异议艺术家"的标签 |
| 不做对抗性行为 | 对视监控不是破坏监控，路牌不是挑战政府命名权 |
| 不做纯商业 | 市集背后的目标是公共连接，不是利润最大化 |
| 不做"领导" | 不把自己定位成社群领袖——"元元的菜地"的名字来自居民，不是他起的 |
| 不做宏大叙事 | 不谈论"改变中国"或"改造社会"——只做具体的事 |

### 3.7 独有贡献：将"直接行动"确立为公共创造的方法论

在 B Community 的所有案例中，葛宇路贡献了一种独特的方法论：

> **直接行动 = 公共创造（Direct Action as Public Creation）**

不是等待公共空间的授权，而是**用自己的行动激活公共性**——你挂一块路牌，整个城市系统会用它的方式回应你；你盯着摄像头，摄像头后面的人会出来和你说话；你办一个市集，不同阶层的人会在同一个地方相遇。

葛宇路的方法论基于一个根本信念：

> "我是想用艺术去加一切，去达成一个很多元的，每个人都有自己的空间的世界。"

**这不是关于"做艺术"，而是关于用艺术打开和别人、和世界连接的方式。** 艺术不是目的，"加一切"才是——艺术是那个用来往固化社会里"加"东西的工具，加连接、加温度、加多样性、加可能性。
