# 绣绣故事会 · 按良好生活标准分类

> **良好生活三问**：
> 1. 是否仍然愿意想象、表达、创造——在既有路径之外，把新的可能带进现实？
> 2. 是否在不确定的现实里，诚实地面对自己，找到内在自洽与韧性？
> 3. 是否在关照自我的同时，关切他人、回应周遭、参与塑造共同世界？

---

## 一、创造力 / Create Well

> *"仍然愿意想象、表达、创造，是在既有路径之外，在线性的成功学叙事之外，并把一些新的可能带进现实。"*

### 1.1 拒绝既有路径：刺绣离开闺阁，走向公共空间

绣绣故事会的创造力起点是**媒介的重新定义**。刺绣在中国传统语境中长期被归类为"女红"——一种私密的、家庭的、装饰性的手工劳动。绣绣故事会没有选择"传承非遗"式的改良路径，而是将刺绣从三个既有的框架中同时解放出来：

| 拒绝对象 | 具体内容 |
|----------|----------|
| 刺绣=女红的传统叙事 | 不将刺绣限定在"手工艺传承"或"非遗保护"的框架内 |
| 刺绣=个人的孤独劳动 | 将刺绣变成集体共创的媒介——"共绣" |
| 刺绣=静止的视觉装饰 | 将刺绣与身体表演结合——披上"许愿池"进行身体再表达 |
| 艺术=白盒子展览 | 不在画廊/美术馆进行，而是在榕树下、社区广场 |

### 1.2 发现新可能：针线作为叙事媒介

绣绣故事会的核心创造在于**将刺绣重新定义为叙事媒介**。针脚不只是图案，而是故事的物质化：

> "每个人都是自己内在空间的建筑师，也是'家'的创造者。"

**"房子编织器"**是这一创造性隐喻的集中体现——将钩编与刺绣的技术转化为关于"家"的心理探索工具。参与者不是在"学习一门手艺"，而是在**用针线建造一个关于自己内在世界的模型**。

**理论根基**：绣绣故事会继承了二高表演对织物的长期探索——从非洲蜡布（African wax print）到刺绣，织物从表演服装转变为独立的创作媒介。这不仅仅是材料实验，而是**对织物自身叙事能力的承认**：每一块布、每一根线、每一个针脚都携带着记忆与故事。

| 创造性维度 | 说明 | 案例 |
|-----------|------|------|
| 技术转译 | 传统女红技术→当代艺术媒介 | 刺绣、钩编作为创作语言 |
| 叙事转译 | 内心愿望→视觉符号→刺绣技术 | "流动许愿池"工作坊流程 |
| 空间转译 | 家庭空间→公共空间 | 榕树下艺术节、甘竹滩广场 |
| 身体转译 | 刺绣成品→身体表演道具 | 披上"许愿池"进行表演再表达 |
| 身份转译 | 手工艺人→共创艺术家 | 最终作品署名"绣绣故事会与所有共创者" |

### 1.3 代表作品：把新可能带进现实

| 作品 | 类型 | 年份 | 创造力的体现 |
|------|------|------|-------------|
| **共绣——流动许愿池** | 集体刺绣+身体表演 | 2024-2025 | 在顺德甘竹滩广场榕树下，将内心愿望转化为视觉符号，再用刺绣技术物化，最后披上"许愿池"进行身体表演——完成从个体心理→视觉符号→物质织品→公共仪式的四重转化 |
| **房子编织器** | 钩编+心理探索工具 | — | 一个DIY教程作品，"每个人都是自己内在空间的建筑师"——用编织技术建造关于"家"的心理模型 |
| **女性与水域** | 空间叙事研究 | — | 对"井边、河边、池边"作为女性劳动中交换故事的传统空间的再发现与当代化 |

### 1.4 设计方法论：绣绣故事会的创造原则

从绣绣故事会的实践中，可以提取出独特的创造方法论：

| # | 原则 | 说明 |
|---|------|------|
| 1 | **慢媒介，深叙事** | 刺绣的慢时间本身就是叙事结构——一针一线、一个故事接一个故事。不以效率为目标，以深度为尺度 |
| 2 | **技术服务于叙事，而非反过来** | 不炫技。钩编、刺绣的技术选择完全服务于"流动的家""许愿池"这些叙事主题 |
| 3 | **集体署名即创造姿态** | "绣绣故事会与所有共创者"——不区分艺术家与参与者，最终的署名本身就是对"艺术=个人天才"叙事的拒绝 |
| 4 | **从内到外的转化链条** | 内心愿望→视觉符号→刺绣技术→集体仪式。每一步都是创造，不跳过任何一步 |
| 5 | **空间的选择即表达** | 榕树下、广场边——不是借场地，而是场地的文化记忆本身就是作品的一部分。榕树在岭南文化中本身就是社区聚会的象征 |
| 6 | **教程而非作品** | "房子编织器DIY教程"发布在B站——开放方法而非保护知识产权。创造不是为了制造稀缺，而是为了扩散可能性 |

### 1.5 命名即创造："绣绣"与"故事会"

名字的两个部分精准定义了实践的两极：

- **"绣绣"**——动作的重复性、时间性、身体性。叠字的温柔与持续，暗示这不是一次性的"创作"，而是持续的"在做"。
- **"故事会"**——集体的、口头的、民间的。不同于"研讨会""分享会"，"故事会"带有一种前现代的、民间的、平等的叙事传统。每个人都有故事，每个故事都值得被倾听。

二者合在一起——**用针线的慢时间，去承载普通人故事的重量**——这就是绣绣故事会最核心的创造。

## 二、身心安顿 / Live Well

> *"如何在不确定、不断变化、也并不总是友好的现实里，依然能够诚实地面对自己，慢慢找到一种内在自洽，同时保有韧性的生活方式。"*

### 2.1 诚实面对：为"流动的一代"造一个家

绣绣故事会面对的不是抽象的艺术命题，而是一个具体的时代处境——**流动的年轻一代找不到"家"的感觉**：

> "流动的家"——为中国当代流动的年轻一代，用集体编织创造一种家的感觉。

这种诚实在于：不假装这个问题不存在，不提供"买房""返乡"式的虚假解决方案，而是**用针线和故事重新定义"家"的含义**——家不是一个物理空间，而是通过共同的劳动和叙事编织出来的关系网络。

### 2.2 内在自洽：免费工作坊的经济伦理

绣绣故事会的工作坊全部免费——没有门票、没有材料费、没有"公益项目"式的行政管理费用。这种经济姿势是一种**内在自洽的选择**：

| 维度 | 内容 |
|------|------|
| 主要支持 | 和地在创（社区艺术支持计划） |
| 活动收费 | 全部免费 |
| 经济逻辑 | 社区艺术支持→免费服务社区参与者。不追求商业变现 |
| 拒绝的资本 | 手工艺培训班的商业模式、非遗文创的商业化路径 |
| 规模定位 | 12人——小规模、深度互动，品质优先于数量 |

这种经济模式的内在自洽在于：**刺绣本身是一个慢媒介，如果追求商业效率和规模化，就从根本上背叛了这种媒介的性质。** 免费不是"缺乏商业模式"，而是对实践本质的诚实承认。

### 2.3 韧性：不子的低调姿态

在三个创始人中，不子没有独立的社交媒体账号。在"个人IP"被奉为圭臬的时代，这是一种罕见的**自我隐身**：

```
不子的低可见度：
├── 无独立社交媒体账号
├── 不进行个人品牌建设
├── 不在作品之外进行自我表达
└── 实践本身即全部存在
```

这种低调不是缺乏能力，而是一种**关于何为"足够"的清醒认知**：不需要个人知名度来验证实践的价值，不需要个人品牌来驱动项目的发展。12个参与者的深度互动比1200个粉丝的数字更重要。

### 2.4 针线的时间伦理：反加速的慢节奏

> 一针一线、一个故事接一个故事。

刺绣作为媒介，天然地抵抗速度。你不能"更快地刺绣"——每一针都需要时间，每一个故事都需要耐心的倾听。这种**媒介自带的慢时间**为绣绣故事会的实践注入了反加速的伦理：

| 时间维度 | 体现 |
|----------|------|
| 工作坊时长 | 需要数小时完成一件作品，不允许匆忙 |
| 故事的发生 | 在刺绣的过程中自然流淌，不需要"引导"或"破冰" |
| 关系的建立 | 12人围坐，针线之间，故事自然交换 |
| 与二高表演的关联 | 不是一次性的快闪，而是在二高表演对织物的长期探索谱系中生长出来 |

### 2.5 价值观作为内在锚点

在绣绣故事会的实践中，可以识别出以下核心价值观：

| 价值观 | 强度 | 如何在生活中体现 |
|--------|:---:|------|
| 人先于利润/数字 | █████ | 全免费、12人小规模、不追求流量和变现 |
| 真实先于表演 | █████ | 不子无个人社交媒体、集体署名、在社区而非画廊空间 |
| 自由先于安全 | ████ | 不依赖商业模式的独立创作路径、依托社区艺术支持 |
| 边缘先于中心 | ████ | 顺德而非广州中心、榕树广场而非美术馆、女性手工艺而非主流当代艺术媒介 |
| 关系先于交易 | █████ | 免费共创、"许愿池"作为心理空间而非商品、故事交换而非信息传递 |

## 三、公共性 / Benefit Well

> *"如何在关照自我的同时，关切他人、回应周遭、关心公共，参与塑造我们共同身处的世界。"*

### 3.1 "共绣——流动许愿池"：公共仪式的基础设施

"流动许愿池"是绣绣故事会公共性的核心实践。它不是一个"作品"，而是一个**让公共仪式得以发生的框架**：

| 维度 | 内容 |
|------|------|
| 形式 | 集体刺绣工作坊+身体表演仪式 |
| 地点 | 顺德甘竹滩广场榕树下——榕树下艺术节 |
| 规模 | 12人，亲密尺度 |
| 流程 | 内心愿望→视觉符号→刺绣技术→集体仪式（披上"许愿池"、身体表演再表达） |
| 精神 | "许愿池"作为心理空间——外化内在愿望，让不可见的内心世界变成可见的共同织品 |
| 门槛 | 免费、零基础可参与——不需要"会刺绣"，只需要"有愿望" |
| 署名 | "绣绣故事会与所有共创者"——集体作者身份 |

**"许愿池"的公共性意义**：在中国传统文化中，许愿池（投币许愿的水池）是一个**公共心理空间**——人们将私人的愿望投入公共的池水中，被陌生人看见又在水中模糊。绣绣故事会将它从水的隐喻转化为织品的隐喻——**愿望不再是沉入水底的硬币，而是可以被披在身上、集体表演、四处流动的织品。**

### 3.2 榕树下艺术节：嵌入社区而非寄生社区

绣绣故事会的工作坊不是孤立的活动，而是**榕树下艺术节**的一部分。这决定了它的公共性不是"把艺术带到社区"的居高临下姿态，而是**成为社区节日的有机组成部分**：

```
传统的"艺术下社区"模式：
艺术家 → 带着作品 → 到社区展览 → 离开

绣绣故事会的模式：
社区节日（榕树下艺术节）
    ├── 当地居民自然聚集
    ├── 榕树的象征（岭南社区聚会的传统空间）
    ├── 绣绣故事会在此开展"共绣"
    ├── 参与者从"社区居民"变成"共创艺术家"
    └── 作品留在社区的记忆中，而非带走
```

**榕树的选择不是偶然的**：在岭南文化中，榕树头下历来是社区集体生活的中心——下棋、聊天、乘凉、讲故事。选择榕树下做刺绣工作坊，是在**激活这个空间原有的记忆和功能**，而非创造一个"艺术事件"。

### 3.3 关系重建的三个层次

绣绣故事会的实践在不同尺度上重建了人与人的连接：

```
层次 1：参与者与自己内心之间
└── "许愿池"作为心理探索工具
    内心愿望→被看见、被命名、被物化→自我觉知
    "每个人都是自己内在空间的建筑师"

层次 2：参与者与参与者之间
└── 12人围坐共绣 → 故事在针线间自然流淌
    井边、河边、池边的女性传统被当代化
    不是"社交"，而是"共事"中的自然陪伴

层次 3：女性手工艺与公共空间之间
└── 刺绣离开闺阁 → 进入榕树广场 → 被看见
    从私密的"女红" → 公共的"共绣"
    从家庭的室内 → 社区的树下
    从个人的劳动 → 集体的仪式
```

### 3.4 "女性与水域"：对传统公共空间的再发现

绣绣故事会提出了一个重要的空间叙事：**"井边、河边、池边历来是女性在劳动中交换故事、建立情谊的场所。"**

| 传统空间 | 功能 | 当代转译 |
|----------|------|----------|
| 井边 | 打水等待中的交谈 | 工作坊中针线之间的交谈 |
| 河边 | 洗衣时的集体劳动 | 围坐共绣的集体劳动 |
| 池边 | 浣纱时的故事交换 | "许愿池"的故事外化与交换 |

**这一发现的公共性意义**：在主流公共空间叙事中，广场是男性的（政治演讲）、茶馆是男性的（谈生意）。而绣绣故事会指出了一条被忽视的女性公共空间谱系——**水域边的劳动空间**。这些空间没有名字、不被记载、不被视为"公共领域"，但确实是女性交换故事、建立情谊、形成支持网络的场所。绣绣故事会将这一传统空间当代化——**在榕树下的广场（另一种"水边"），让刺绣重新成为女性（及其他性别）共同劳动、共同叙事的媒介。**

### 3.5 开放共享：方法论的扩散

绣绣故事会的公共性不止于工作坊的现场，更通过开放共享将方法论扩散出去：

| 共享渠道 | 内容 | 公共性意义 |
|----------|------|-----------|
| B站"房子编织器DIY教程" | 将核心创作方法免费公开 | 任何人可以学习、改编、再创造——不是"看我的作品"，而是"你也可以做" |
| Instagram @xiuxiugushihui | 作品记录、过程分享 | 27条评论的工作坊招募帖——比许多大号更深度的社区互动 |
| 作品巡回展出 | 从顺德榕树到其他展场 | "许愿池"作为可移动的公共仪式——在不同的空间承载不同人群的愿望 |

**27条评论的意义**：在流量驱动的社交媒体生态中，一个有影响力的账号可能收到大量但浅层的点赞。而一个招募帖子能引发27条评论，意味着**深度参与而非广泛曝光**——这不是"传播广"，而是"连接深"。

### 3.6 边界与选择：什么不是公共性

绣绣故事会的公共性是有边界和有意识的：

| 不做的事 | 原因 |
|----------|------|
| 不做商业手工艺培训 | 刺绣不是技能交易，而是叙事媒介 |
| 不做非遗传承式的保护 | 不是让刺绣"活下去"，而是让刺绣"做新的事" |
| 不在画廊/美术馆做 | 刺绣需要回到它原本所在的社区空间 |
| 不做个人艺术家品牌 | 集体署名、"绣绣故事会与所有共创者" |
| 不做大规模工作坊 | 12人——再大就会失去"井边交谈"的亲密性 |

### 3.7 独有贡献：将"女性手工艺"确立为公共叙事媒介

在 B Community 的所有案例中，绣绣故事会贡献了一种独特的方法论：

> **刺绣作为公共叙事媒介（Embroidery as Public Narrative Medium）**

不是将刺绣作为"文化遗产"来保护，不是作为"手工艺产品"来开发，也不是作为"个人艺术创作"来署名——而是**将刺绣拉到公共空间，让普通人的故事和愿望通过针线获得物质形态，再通过集体仪式被表演和流动。**

**不子的实践暗示**：无个人社交媒体——刺绣不是"我的作品"，而是"我们共同的故事"。这种去个人化的姿态，恰恰是让刺绣从"艺术家媒介"回到"民间媒介"的关键一步。
