# 廖智立 · 按良好生活标准分类

> **良好生活三问**：
> 1. 是否仍然愿意想象、表达、创造——在既有路径之外，把新的可能带进现实？
> 2. 是否在不确定的现实里，诚实地面对自己，找到内在自洽与韧性？
> 3. 是否在关照自我的同时，关切他人、回应周遭、参与塑造共同世界？

---

## 一、创造力 / Create Well

> *"仍然愿意想象、表达、创造，是在既有路径之外，在线性的成功学叙事之外，并把一些新的可能带进现实。"*

### 1.1 理论创造："普罗托邦"作为新的社会想象框架

廖智立的创造力首先体现在**概念层面的创新**。"普罗托邦"（Protopia）不是他发明的词，但他将其系统性地引入中国社会创新语境，赋予它明确的实践内涵：

> 普罗托邦不是乌托邦——不追求一个完美的、终结性的理想社会；也不是反乌托邦——不陷入绝望和犬儒。它是"渐进但持续地让世界变好一点"的实践哲学。

在"要么完美理想、要么彻底绝望"的二元叙事之外，普罗托邦提供了一种**第三条道路**：接受世界永远不会完美，但每一天都可以比昨天更好一点。这种理论姿态本身就是在创造新的可能——改变人们"想象社会变革"的方式。

| 创造维度 | 具体内容 |
|----------|----------|
| 概念译介 | 将"Protopia"概念系统引入中文社会创新话语 |
| 理论落地 | 不是停留在哲学讨论，而是将其与共居实践绑定 |
| 叙事重构 | 打破"乌托邦 vs 反乌托邦"的二元对立 |
| 知识生产 | "2025 共居生活洞察报告"——从实践中提炼可传播的知识 |

### 1.2 实践创造：共居作为创造性的生活设计

廖智立的创造不止于理论。他将"共居"（co-living）从一种居住方式提升为一种**设计实践**：

| 实践维度 | 内容 | 创造性的体现 |
|----------|------|-------------|
| 空间设计 | 参与 Dweller 青年共居社区的营造 | 不是"租个房子住"，而是设计一种共享生活的空间规则 |
| 社群营造 | 连接 706、GCC Commons 等共居网络 | 让孤立的共居实验变成可连接的生态 |
| 知识生产 | "2025 共居生活洞察报告" | 将零散的共居经验系统化为可传播的知识 |
| 生活实验 | 自身住在实验性社区中 | 不是在"研究"共居，而是在"实践"共居 |

### 1.3 SSIR 翻译：跨世界的转译者

廖智立作为《斯坦福社会创新评论》（SSIR）中文版的译者，其创造力体现在一种特殊的**转译能力**上：

> 他不只是翻译语言（English → 中文），而是在翻译"世界"——将全球社会创新的理论资源，转译为中文读者可以理解、可以使用的知识。

这种翻译工作是创造性的，因为它需要：理解原始语境、识别中国读者的知识缺口、找到合适的语言让陌生的概念变得可感可用。

| 翻译作为创造性实践 | 说明 |
|----------|------|
| 语言转译 | 英文 → 中文，准确且可读 |
| 概念转译 | 全球社会创新话语 → 中国在地语境 |
| 领域转译 | 学术理论 → 社区实践者的日常语言 |
| 知识开通 | 让原本只属于英文读者的知识资源，对中文读者开放 |

### 1.4 创造方法论：在"之间"工作

从廖智立的实践中，可以提取出他的创造姿态——**他总是在"之间"工作**：

| # | "之间" | 说明 |
|---|------|------|
| 1 | **理论与生活之间** | 普罗托邦不是书斋里的哲学，是每一天都在实践的生活方式 |
| 2 | **个人与社群之间** | 共居既是个人的居住选择，也是社群的共同创造 |
| 3 | **全球与本地之间** | SSIR 翻译将全球知识接入中国在地实践 |
| 4 | **写作与行动之间** | 报告和翻译不是"做完就算"，而是行动者使用的工具 |
| 5 | **理想与现实之间** | 普罗托邦就是"在理想与现实的缝隙中持续做事" |

这种"之间"的位置不是摇摆不定，而是**一种方法论选择**——不站任何一个极端，而是在连接处创造价值。

## 二、身心安顿 / Live Well

> *"如何在不确定、不断变化、也并不总是友好的现实里，依然能够诚实地面对自己，慢慢找到一种内在自洽，同时保有韧性的生活方式。"*

### 2.1 "普罗托邦"作为心理锚点

廖智立的"身心安顿"与他创造的"普罗托邦"概念高度一体。普罗托邦不只是一种社会理论，也是一种**个人如何与世界相处的内在姿态**：

> 接受世界不会完美，但每一天都有可能让一小部分变好一点。

这种哲学在心理层面的功能是：它同时抵御了两种常见的心理陷阱——**乌托邦的幻灭**（"我们永远达不到理想社会，所以一切都没意义"）和**反乌托邦的犬儒**（"世界只会越来越糟，做什么都没用"）。普罗托邦给出的答案是："确实不完美，但做一点是一点"——这是一种**诚实的希望**。

| 普罗托邦的心理维度 | 内容 |
|----------|------|
| 对抗幻灭 | 不追求完美终点，所以不会因为达不到而崩溃 |
| 对抗犬儒 | 不相信彻底绝望，所以始终有做事的动力 |
| 接受渐进 | 不以"是否彻底改变世界"来衡量行动的意义 |
| 拥抱不完美 | 在"还不够好"的状态里依然能安顿自己 |

### 2.2 知行合一：用居住方式实践自己的哲学

廖智立不是在理论上倡导共居，而是**自己住在实验性的共居社区里**。这种选择本身就是身心安顿的体现——他没有住在"正常"的房子里，然后写文章说"共居很好"。他的知识工作（翻译、研究、写报告）和他的生活实践（住在共居空间、参与社区营造）是同一件事的两个面。

这种一致性具有重要的心理意义：**不需要分裂**——不需要在工作中说一套、在生活中做另一套。理论喂养生活，生活检验理论。

### 2.3 韧性：在不确定中的持续在场

廖智立的工作领域——替代性居住方式、青年共居实验——在中国社会环境中是一个充满不确定性的领域。政策模糊、公众认知有限、经济模型不稳定。在这样的条件下持续做这件事，本身就是韧性的表现。

普罗托邦哲学为这种韧性提供了内在支撑：如果目标是"彻底改变住房制度"，那确实容易绝望；但如果目标是"让一小群人今天住得更好一点"，那每一天都可以是成功的。

### 2.4 价值观作为内在锚点

| 价值观 | 强度 | 如何在生活中体现 |
|--------|:---:|------|
| 渐进优于完美 | ████ | 普罗托邦哲学——不等待完美方案，能做一点是一点 |
| 实践先于宣言 | ████ | 住在共居社区里而非只是研究共居 |
| 连接优于孤立 | ████ | 将不同共居网络的人与知识连接起来 |
| 知识作为公共资源 | ████ | 翻译和报告不是为了个人品牌，是为了让更多人可以用 |
| 诚实面对局限 | ████ | 普罗托邦本身就是对"人类能力有限"的诚实承认 |

## 三、公共性 / Benefit Well

> *"如何在关照自我的同时，关切他人、回应周遭、关心公共，参与塑造我们共同身处的世界。"*

### 3.1 让共居知识成为公共资源

廖智立公共性贡献的核心是**将共居经验转化为公共知识**：

| 知识生产 | 内容 | 公共性意义 |
|----------|------|-------------|
| "2025 共居生活洞察报告" | 对中国共居实验的系统性记录和分析 | 让零散的共居实践变得可见、可言说、可学习 |
| SSIR 中文翻译 | 全球社会创新知识的汉化 | 打破语言壁垒，让中文读者接触前沿社会创新 |
| 社区文档 | Dweller 等共居空间的实践记录 | 让一个社区的经验可以服务更多社区 |

这些知识不是锁在学术期刊里的，而是面向实践的——**为正在做共居或想做共居的人提供地图和工具**。

### 3.2 连接基础设施：编织共居网络

廖智立不只是住在 Dweller，他是**不同共居网络之间的连接点**：

```
       706 青年空间
           |
    GCC Commons ──── 廖智立 ──── Dweller 青年共居
           |
        其他共居实验
```

他不是在"建立自己的品牌"，而是在**让已有的不同节点之间产生连接**。这种连接工作容易被忽视——它不产出"自己的作品"，但它让别人的作品能够互联、交换、相互滋养。

| 连接的维度 | 内容 |
|----------|------|
| 空间连接 | 让 Dweller、706、GCC Commons 等共居空间互通 |
| 知识连接 | 全球社会创新理论 ↔ 中国在地实践 |
| 人群连接 | 研究者 ↔ 实践者 ↔ 想了解共居的普通人 |

### 3.3 翻译作为公共服务

廖智立的 SSIR 翻译不是商业行为，而是一种**公共服务**：

> 社会创新领域的大量核心文献和前沿讨论以英文为主。对于不会英文的中国实践者，这些知识是不可及的。翻译就是在打开这扇门。

这不是"用自己的语言表达自己的思想"（那是创作），而是"用自己的语言让别人接触别人的思想"（那是一种知识公地的维护）。在一个鼓励每个人都做"思想领袖"的时代，选择做一个**翻译者**——让他人的思想可以被更多人看见——是一种低调但关键的公共实践。

### 3.4 为替代性居住方式建立合法性

在中国社会，替代性居住方式（共居、合作居住、青年社区）仍然处于主流认知的边缘。廖智立的知识生产——研究报告、翻译文章、社区记录——是在为这些实践**建立合法性**：

| 合法性的层次 | 如何建立 |
|----------|------|
| 认知合法性 | 报告和翻译让公众知道"共居是什么" |
| 实践合法性 | 系统性的记录让共居实践者看到"这件事是有人在认真做的" |
| 政策合法性 | 严谨的研究和记录为未来可能的政策讨论提供基础材料 |

### 3.5 边界与选择

| 不做的事 | 原因 |
|----------|------|
| 不做乌托邦式的宏大宣言 | 普罗托邦本身就是对"完美理想"的拒绝 |
| 不把共居包装成生活方式消费品 | 共居是生活实践，不是一种"潮流" |
| 不把知识锁在学术圈 | 报告和翻译面向实践者，而非学术评审 |
| 不追求个人品牌 | 连接者而非明星——把光打在共居实践本身 |

### 3.6 独有贡献：将"替代性居住"确立为社会创新的场域

在 B Community 的案例中，廖智立贡献了一个尚未被充分认识的维度：

> **共居（co-living）作为社会创新的实验场**

共居不只是"几个人住在一起省钱"，而是：重新设计人与空间的关系、重新定义"家"的边界、重新实践日常生活中的合作与协商。通过他的翻译、研究和网络编织，廖智立让这个实验场变得可见，让其中产生的经验和知识可以流动到更广阔的社会创新领域。
